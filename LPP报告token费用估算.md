# LPP 报告 Token 费用估算

> 基于 DeepSeek API 定价（2026-05）

## 每次分析的 Token 消耗

| 项目 | 大小 |
|------|------|
| LPP 技能框架（SKILL.md + 核心参考） | ~40K tokens |
| 用户提交文本（平均 600 字） | ~400 tokens |
| **输入合计** | **~40.4K tokens** |
| **pro 报告输出** | **~2,000 tokens** |
| **lite 版输出** | **~800 tokens** |

## 费用明细（DeepSeek API）

- 输入：¥2 / 1M tokens
- 输出：¥8 / 1M tokens

### Pro 版

| 场景 | 计算 | 金额 |
|------|------|------|
| 无缓存（首份） | 40.4K×¥2 + 2K×¥8 | **¥0.10** |
| 有缓存（后续） | 0.4K×¥2 + 2K×¥8 | **¥0.025** |

### Lite 版

| 场景 | 金额 |
|------|------|
| 无缓存 | **¥0.09** |
| 有缓存 | **¥0.01** |

## 建议定价

成本极低（几分到一毛），建议：
- **象征性收费 ¥1-5/份**，覆盖成本 + 筛选意愿
- 或 **免费开放**，成本可忽略

## 备注

- 框架缓存：多人先后提交时，框架内容在前一次已加载，后续仅需付用户文本 input + 报告 output
- 费用基于 DeepSeek 官方公开定价，实际使用可能因上下文长度变化有轻微浮动
