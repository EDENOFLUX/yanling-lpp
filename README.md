# 言灵 LPP

**Linguistic Psychological Pattern** — 从语言模式分析心理特征的语言心理学 AI skill。

## 简介

言灵 LPP（言霊/kotodama）是一个基于语言模式分析的心理特征推理框架。通过扫描文本的 40 个语言维度（D1-D40），结合神经多样性校准（ND层），定位到 46 个社交场景光谱，最终归纳出大七人格维度评分和心理画像。

## 安装

将此 skill 放入 `~/.claude/skills/` 目录，或在 Claude Code 中直接引用。
