# LPP 场景式表达光谱框架 — Scene-Based Expression Spectrum

## 概述

本框架采用自下而上（bottom-up）的方法论：以"同一交际意图下不同人的不同表达方式"为分析单位，构建双极光谱。每个光谱从1到10代表从一端到另一端的连续过渡，每个刻度位置标注该位置人群的典型语言标记和表达示例。

### 方法论原理

大五人格最初由Allport整理 trait words → Cattell缩减 → Norman/Fiske因子分析而诞生。本框架沿同样思路：**不从理论出发，而从表达差异出发**。每个场景自包含、不预设理论标签，仅在光谱两端用行为描述（而非心理学诊断术语）来定义。

### 使用方式

1. 拿到文本后，定位文本中的"交际场景"（对方在做什么？表达欣赏？表达拒绝？表达诉求？）
2. 对每个命中的场景，将对方措辞与光谱对照，定位其在光谱上的位置
3. 跨场景的定位一致性 → 稳定的表达风格（即"特质"）
4. 跨场景的定位分歧 → 情境敏感度（本身是一个独立的信息维度）

## 场景一：表达欣赏/赞美

### 光谱A：赞美的方式 — 具体 vs 笼统

| 位置 | 语言标记 | 示例 |
|------|---------|------|
| 1-2 具体叙事型 | 你记得吗那次你…我当时就觉得… | "你还记不记得大三那次pre，你上去第一句话就把全场镇住了" |
| 3-4 具体归因型 | 你XX做得很好，因为… | "你那个方案做得特别好，切入点很准" |
| 5-6 笼统品质型 | 你真好/你太厉害了/你很棒 | "你真的很优秀""你太靠谱了" |
| 7-8 夸张渲染型 | 太…了/绝了/无敌/封神 | "你也太强了吧！！直接封神！" |
| 9-10 空洞社交型 | 不错/挺好/可以 | "不错不错""挺好" |

### 光谱B：赞美的锚点 — 能力 vs 人格 vs 结果

| 位置 | 语言标记 | 示例 |
|------|---------|------|
| 1-2 赞能力 | 你太聪明了/你能力很强 | "你学习能力真的很强" |
| 3-4 赞人格 | 你太温柔了/你真的很善良 | "你是一个特别有耐心的人" |
| 5-6 赞结果 | 你这个做得真好/效果很棒 | "这篇文章写得真好" |
| 7-8 赞品味/眼光 | 你眼光真好/你太会了 | "你挑礼物的品味绝了" |
| 9-10 综合型 | 你这个人做什么都… | "你真的是做什么都能做好的人" |

### 跨场景整合分析指南

### 模式识别

当分析对象在多个场景中呈现一致的光谱位置时，这些一致性构成稳定的表达风格（"特质"）：

| 跨场景一致性 | 可能的心理含义 |
|-------------|---------------|
| 多处场景均偏具体端（1-4） | 具体化认知风格，偏好精确沟通 |
| 多处场景均偏笼统端（7-10） | 概括化认知风格，可能高开放性/低精确需求 |
| 多处场景均偏直接端（1-4） | 低社交监控，高真实性偏好 |
| 多处场景均偏回避端（7-10） | 高社交监控，高冲突回避 |
| 情绪场景偏向压抑端 | 情绪管理倾向，可能的低情绪表达性 |
| 关系场景偏向焦虑端 | 关系焦虑倾向 |

### 情境敏感性分析

当同一人在不同场景中处于光谱的不同位置时，这本身是有信息量的：

| 分歧类型 | 可能的含义 |
|---------|-----------|
| 专业领域自信 vs 社交领域谦虚 | 选择性自信心，与领域安全感相关 |
| 对亲密关系直接 vs 对外人回避 | 高关系辨别力，正常的社交边界 |
| 对上级恭敬 vs 对下级傲慢 | 高权力距离敏感，可能的权威人格倾向 |
| 公开场合克制 vs 私下表达强烈 | 高自我监控，印象管理能力强 |

### 使用限制

1. 每个场景光谱是独立的分析工具，不预设任何特定人格理论
2. 光谱位置本身无好坏，"高分"不代表"更健康"或"更不健康"
3. 同一位置在不同语境下可能有不同含义（对上级说"随便"vs对伴侣说"随便"）
4. 跨场景的一致性需要至少3-5个场景的覆盖才能得出
5. 单场景的定位只能作为一个观察点，不能作为特质推断依据

## 跨场景密度光谱（D22 信息密度的Layer 2应用）

D22作为Layer 1维度评分后，可在Layer 2中作为**跨场景密度光谱**应用于所有45场景。

### 分析方法

```
场景A ████████░░  高密度     → 该场景中信息密度高（精炼高效）
场景B ████░░░░░░  低密度     → 该场景中信息密度低（冗长口语化）
场景C ██████░░░░  中等密度   → 该场景中密度适中
```

### 跨场景密度分布的心理映射

| 密度分布模式 | 可能的心理指向 |
|------------|---------------|
| 所有场景密度一致（波动<15%） | 稳定的认知风格，不受社交情境影响——可能是低社交监控或极稳固的思维习惯 |
| 正式场景高密 + 亲密场景低密 | 正常的社交调节能力，高情境敏感度 |
| 对特定话题突然变高密 | 该话题是认知舒适区/专业领域 |
| 对特定话题突然变低密 | 该话题触发情绪/不确定感，或是有意放松关系距离 |
| 所有社交场景均低密（无高密场景） | 口语化认知风格，高外向性，偏好关联型表达 |
| 所有社交场景均高密（无低密场景） | 书面型认知风格，严谨偏好，可能的社交距离感 |

### 在场景报告中的呈现

```
─── 跨场景密度分析（D22）───

表达欣赏 ████████░░  高密度     → 精炼归因型
表达感激 ████████░░  高密度     → 具体归因型
告别收尾 █████░░░░░  中密度     → 仪式化但不过度
表达亲密 ████░░░░░░  低密度     → 放松、口语化
表达幽默 ████░░░░░░  低密度     → 轻松、填充词多

整体高密:低密 = 35%:25% → 均衡型，随场景调节
模式解读：在需要表达深度时收紧密度（归因式连接），
在放松社交时释放密度（填充词增多）。这是典型的
"认知精炼+社交放松"双模式——密度本身不是固定值，
而是你的社交调节工具。
```

## D30-D40 跨维度观察→人格映射表（补充归纳路径）

以下表格将D30-D40各观察维度直接映射至大五/大七人格空间，作为场景映射表的**补充路径**。D30-D40维度的定位不依赖特定交际场景，而是从文本整体的语言形式模式中提取——它们衡量的是"怎么说"的风格特征（情态策略/叙事风格/完结模式/评价框架），而非"在什么场景下说了什么"。

**与场景映射表的关系**：
- 场景映射表（场景→大五/大七）：主要路径，精度高，依赖特定场景命中
- D30-D40映射表（补充路径）：辅助路径，覆盖场景未命中的维度方向，当场景映射证据稀疏时提供额外信号
- **融合规则**：D30-D40的信号不单独主导人格归纳，而是在场景映射证据不足（某维度仅0-1个场景命中）时作为辅助证据参与加权聚合。当D30-D40信号与场景映射信号方向一致时→增强场景归纳置信度（+0.5场景权重的等效）；方向冲突时→不抵消场景归纳，仅标注"维度内部存在跨路径分歧"。

**权重规则**：
- D30-D40映射的默认权重为**情境参考（权重=1）**——精确度低于场景定位
- 当某维度在≥3个不同段落中有高度一致的信号时，权重升至**次要指标（权重=2）**
- 当某维度仅1处引用时，不参与聚合（仅标注为探索性观察）

**评分方法**：
- 先定位该D维度在文本中的表现倾向（偏向1-3端还是8-10端）
- 根据端向和映射维度，在1-10区间给出该维度的信号值
- 信号值计入对应人格维度的加权聚合（与场景映射的评分规则相同）

### 使用说明

1. **定位方法**：先判断该D维度在文本整体上偏向1-3端还是8-10端，参考维度定义（详见 `indicators.md` 中D30-D40的观察指南）
2. **跨段落一致性**：同D维度在≥3个不同段落中信号方向一致 → 权重升至 2（次要指标）；仅1-2处 → 权重保持 1（情境参考）
3. **与场景映射的融合**：当某人格维度在场景映射中无命中场景时，D30-D40映射可作为主要归纳路径（权重升至2）；当场景映射已有≥2个场景命中时，D30-D40仅做辅助验证
4. **跨路径分歧处理**：同一人格维度在场景映射和D30-D40映射中方向相反 → 以场景映射为准，在报告第二部分标注"该维度的场景定位与跨维度观察存在分歧"
5. **大七T维度的补充路径**：T（处世态度）在大五中无直接对应，D36（认知谦逊）和D40（评价锚定）是T维度在D30-D40映射中的主要信号源——当场景映射未覆盖T维度时，优先使用这两个维度的D30-D40信号
