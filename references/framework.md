# LPP 理论框架

本文档提供LPP分析流程的理论基础：

- **Step 1（扫描）的理论基础**：为什么这些语言模式有心理信息量——每个观察维度（D1-D40）的研究来源和局限性
- **Step 1.5（ND校准）的理论基础**：神经多样性特征的语言标记及对人格归纳的confound机制
- **Step 2（场景）的理论基础**：为什么同一交际意图下的措辞差异反映心理差异
- **Step 3（归纳）的理论基础**：如何从跨场景模式归纳人格（中文→大七 / 欧美→大五）和依恋风格，交叉验证逻辑
- **可选输出层的理论基础**：GFP聚合、暗黑人格检测、LLM验证层、零样本LLM推估验证

> ⚠️ **重要方法论说明**：
> LPP使用多步流程：Step 1只做语言模式扫描（观察），Step 1.5做神经多样性confound校准，Step 2做场景光谱定位（定位），Step 3才做心理归纳（解释）。
> 本文档的所有"心理映射"内容仅作为Step 3归纳时的参考依据，不应在Step 1和Step 1.5中使用。
> 关于各观察维度的详细扫描指南（具体看什么、怎么记录），见 `references/indicators.md`。

## ⚠️ 重要方法论：方言/口音干扰识别

**口音和方言不是人格特征。** 在评分之前，必须区分：
- **个体选择**（反映心理特征的语言习惯）
- **群体特征**（反映地域/社群/母语背景的语言特征）

### 需要做方言校正的常见情况

| 语言特征 | 可能被误读为 | 实际可能是方言影响 |
|---------|-------------|-----------------|
| 语气词"撒""咯""噻"密集 | 高亲和需求(D6) | 四川/湖南/重庆方言特征 |
| 语气词"啦""喔""捏"密集 | 高亲和需求(D6) | 台湾/闽南语区特征 |
| "您"使用极少 | 低权力距离(D14) | 南方方言区惯例（粤语/吴语等无敬称二分） |
| "您"使用极多 | 高权威敏感(D14) | 北方方言区惯例（北京/天津等） |
| 句末语气上扬/特定填充词 | 不确定/犹豫(D5/D6) | 地域性语调特征（如东北'整'、天津'嘛'） |
| 特定句式（如"你有吃饭吗"） | 语法异常 | 粤语/闽语语法迁移至普通话 |
| 标点偏少/不断句 | 低严谨性(D1) | 口语习惯迁移、方言书写经验少 |
| 极度正式的书信体 | 高权威敏感(D14) | 台湾/香港的正式书写传统 |

### 校正原则
1. **识别 → 标注 → 降权**：先识别可能的方言特征 → 标注为confound → 降低该特征在心理评分中的权重
2. **一致性检查**：如果某个特征在方言区人群中普遍存在，则个体出现该特征不具心理区分力
3. **跨样本对比**：同一人在方言场景 vs 普通话场景下的语言差异，本身是有信息量的（反映语码切换能力）
4. **不确定时写进报告**：如果无法判断是方言还是人格特征，在报告中标注"该特征可能受方言影响"

## ⚠️ 工作语境校准层（Work Context Confound）

**工作文本的特殊性**：一个人在职场中的表达受到三重约束——职业角色（扮演"专业人士"）、权力结构（上下级/平级关系）、结果预期（表达有职业后果）。这使得工作文本成为"高面具"样本，大量语言模式反映的是角色规范而非人格特质。

### 工作文本 vs 自然表达的基线偏移

以下D维度在工作语境中存在系统性偏移，纳入Step 3归纳前必须校准：

| D维度 | 工作语境典型偏移 | 可能被误读的方向 | 校准规则 |
|-------|----------------|-----------------|---------|
| D6 语气词 | **↓↓ 显著降低**（趋零） | 低亲和需求/高正式度 | 工作语境中D6低频→不纳入E/R评分；降为▲弱信号 |
| D9 情绪词 | **↓↓ 显著降低**（压抑） | 情绪稳定/低N | 不可做N低分证据；工作语境中情绪词稀疏→标注▲confound |
| D10 绝对化词汇 | **↓ 降低**或转为修饰性（"某种程度上"） | 认知灵活性高/低自恋 | 不反映真实O或自恋倾向；工作语境绝对化低→不纳入评分 |
| D11 否定词 | **↓ 委婉化**（"可能不太合适"替代"不行"） | 低对抗性/高宜人性 | ≠真实冲突处理风格；如需判断→权重降为0.5 |
| D8 人称 | **↑ "我们"增多，"我"减少** | 集体主义/低自我凸显 | ≠真实个人-集体平衡；"
我们"密集仅标记不计入R/I评分 |
| D15 正式度 | **高且稳定**（职业规范强制） | 高权力距离敏感/高尽责 | 工作语境正式度高→不纳入A/T评分判断 |
| D24 不确定性回避 | **↑ 策略性模糊增加**（职业委婉语） | 模糊容忍度高/低决断力 | ≠真实O或N；不纳入不确定性相关判断 |
| D37 动作-状态动词 | **↑ 动作动词偏多**（工作语言要求结果导向） | 行动导向/高I | ≠真实行动偏好；工作语境动作动词多→不纳入I评分 |
| D38 正当化密度 | **↑ 结构化堆叠**（职业沟通要求"有依据"） | 防御性高/高C | ≠真实防御水平；堆叠理由在工作语境中→降为▲弱信号 |
| D2 句号/断句 | **规范可控** | 条理性强/高A | 工作语境中规范断句→不纳入A评分 |
| D5 断裂句/自我修正 | **↓↓ 极少**（表达经修饰） | 思维连贯/高I | ≥2处断裂方可入场景定位 |
| D16 承诺语言 | **↑ 具体度策略性提高**（职业承诺有后果） | 高尽责 | 工作承诺≠人格承诺；需与工作外承诺语言对比 |
| D21 表达节奏 | **节奏均匀、变化小** | 情绪稳定/低波动 | ≠真实情绪调节方式 |

### 工作面具指数（Work Mask Index）

工作语境本身构成一层"职业面具"，独立于标准面具厚度指数：

| 工作文本占比 | 工作面具系数 | 对置信度的影响 |
|------------|------------|--------------|
| **纯工作文本**（100%职业场景） | ×1.3 | 标准面具厚度指数上浮30%，置信度×0.7 |
| **工作主导**（≥70%工作场景） | ×1.2 | 面具厚度指数上浮20%，置信度×0.8 |
| **工作混合**（30-70%工作场景） | ×1.1 | 面具厚度指数上浮10%，置信度×0.9 |
| **少量工作**（<30%工作场景） | ×1.0 | 不额外调整 |

**工作面具校准方法**：
```
校准后面具厚度 = 标准面具厚度指数 × 工作面具系数
                 ↓
工作样本特有的"角色性面具"与"个人面具"叠加
```

### 工作语境的特殊归纳限制

当样本来**全部或主要来自工作语境**时，以下归纳维度需附加限制：

1. **E（外向善良/大七）/ E（外向性/大五）**：工作语境中的人际亲和可能是职业教养而非真实热情——"礼貌微笑≠善良"。**该维度评分上界默认降低1分**（上限从10→9），标注"疑似高面具影响"。

2. **T（处世态度/大七）**：职场中表达"淡泊"可能只是职业边界感（不谈个人价值观），而非真正的随遇而安。**评分置信度降一级**。

3. **R（人际关系/大七）/ A（宜人性/大五）**：工作中的"好相处"可能是职业需要（团队协作要求）而非真实信任倾向。**需要≥2个跨场景冲突端证据（如对同事表达过明确不信任）才可做低分判断**。

4. **依恋风格**：工作文本极少包含亲密关系场景——**依恋归纳通常不可靠**。标记为△探索性，措辞使用"有限证据提示"。

### 工作语境 + ND confound 交互效应

工作语境与某些ND标记存在交互作用，需双重校准：

| ND标记 | 工作语境下的特殊表现 | 交互校准规则 |
|--------|-------------------|-------------|
| ND4（masking标记） | 工作中masking是常态——ND4↑在工作文本中区分度低 | 工作语境中ND4↑权重降半级（■→●，●→▲） |
| ND3（注意力切换） | 工作文本的结构化要求可能压制ADHD式的跳跃 | ND3●即使在低话题跳跃的工作文本中也不取消激活；标注"工作语境可能mask了注意力切换特征" |
| ND6（时间盲） | 工作语境强制时间管理可能掩盖时间盲 | ND6仅在工作文本中出现≥2处明显"时间管理失败"描述时才激活；缺省状态→不判断 |

### 混合样本优先规则

当样本既包含工作文本又包含个人文本时，以个人文本部分的观察为主，工作文本为辅：
- **个人文本中的模式权重 = 1.0**（全值使用）
- **工作文本中的模式权重 = 0.5**（半值使用，标注confound）
- 工作文本中观察到的模式如果与个人文本一致 → 可以增强置信度
- 工作文本中观察到的模式如果与个人文本矛盾 → 以个人文本为准，工作文本模式标注"推测为职业角色影响"

### 工作语境判定流程（Input Preparation阶段）

```
1. 评估文本来源：是否明确来自工作场景（邮件/会议记录/工作聊天/周报/汇报等）？
2. 估算工作文本占比：工作场景内容占全文百分比
3. 记录工作角色性质：管理者/执行者/独立贡献者/客户面对？不同角色面具厚度不同
   - 管理者/领导角色 → 面具系数额外+0.1
   - 客服/销售/对外岗位 → 面具系数额外+0.05
   - 技术/独立贡献 → 面具系数无额外加成
4. 输入Step 1时标记工作文本段落
5. Step 2场景定位时不将工作场景中的"职业表达"作为稳定人格证据
6. Step 3归纳时应用上述校准规则
```

## ⚠️ 其他高面具/高confound语境校准层

除工作语境之外，以下语境同样存在系统性 confound，需在 Input Preparation 阶段识别后应用对应校准。

---

### ❶ 面试/评价语境（Interview / Evaluation）

**confound机制**：工作语境的极限版本——角色脚本更强（"面试者""被评价者"剧本），评价压力直接决定结果，印象管理动机最大。这是最接近"完全戴面具"的状态。

**基线偏移**：

| D维度 | 面试/评价语境偏移 | 校准规则 |
|-------|-----------------|---------|
| D6语气词 | 趋零（高度控制） | 完全排除，不纳入任何评分 |
| D9情绪词 | ↓↓ 极低（控场） | 不纳入N评分 |
| D10绝对化 | ↓↓ 极少（怕说死） | 不纳入O/自恋判断 |
| D24不确定性 | ↑↑ 策略性模糊大量增加 | 不纳入O/N判断 |
| D38正当化 | ↑↑ 堆叠式（结构化为答案） | 仅标记不纳入评分 |
| D5断裂句 | ↓↓ 几乎为零（事先准备） | ≥3处才标记 |
| D37动作动词 | ↑↑（STAR法则等模板） | 不纳入I评分 |
| D15正式度 | 极高 | 不纳入A/T判断 |

**系数调整**：
- 标准面具厚度指数 × 1.5（比工作更强）
- 置信度 × 0.5
- 面试内容不纳入依恋归纳（关系场景缺失）
- 可直接引用工作语境校准层中的ND交互规则

**特殊限制**：面试文本不可用于任何"归因风格"判断——面试者会自动将成功归因于自己、失败归因于外部，这是面试策略而非人格。

---

### ❷ 公开社交媒体语境（Public Social Media — LinkedIn / 微博大号 / 加了同事的微信朋友圈）

**confound机制**：audience 混合——同事、同行、陌生人、亲友同时在场。表达经过自我审查，任何"不合适的"内容被删除。与工作语境不同：没有即时权力结构，但有**想象中的观众评价**和**个人品牌建设**动机。

**基线偏移**：

| D维度 | 公开社媒语境偏移 | 校准规则 |
|-------|----------------|---------|
| D6语气词 | 中-低（介于社交和正式之间） | 权重降半 |
| D8人称 | "我"增多（个人品牌建设） | ≠真实自我凸显；标记但不纳入R/I评分 |
| D9情绪词 | 校正偏正（展示积极） | 负面情绪缺失→不纳入N判断 |
| D10绝对化 | ↓ 降低（怕被喷） | 不纳入O判断 |
| D15正式度 | 中-高（视平台和受众） | 需与私人消息对比后才有信息量 |
| D35评价框架 | 对他人的评价倾向中立/正面（公开得罪人不划算） | 不作为R/A评分依据 |
| D36认知谦逊 | ↑ 策略性谦逊（"还在学习中""抛砖引玉"） | 不反映真实认知谦逊水平 |

**系数调整**：
- 面具厚度指数 × 1.2
- 置信度 × 0.75
- 仅当有≥3条不同时间/不同话题的帖文时才可做稳定模式判断
- 评论区互动比主帖更有信息量（减少策划成分）

**ND交互**：ND4（masking）在社媒语境中普遍激活且区分度低，权重降半。ND3（注意力切换）在社媒帖文之间的断裂是平台常态，不视为注意力特征。

---

### ❸ 早期亲密关系语境（Early-stage Romantic Relationship — 约会期/追求期/刚在一起）

**confound机制**：展示"最佳自我"的经典场景——负面信息被抑制，配合性表达增多，分歧被软化或回避。这是**印象管理动机最高但角色脚本最模糊**的语境——没有既定剧本但渴望被喜欢，因此表达的策略性反而更高。

**基线偏移**：

| D维度 | 早期亲密关系偏移 | 校准规则 | 可能误读方向 |
|-------|----------------|---------|-------------|
| D9情绪词 | ↑ 正面情绪词上升，负面↓ | 不基于正面密度判断高E/高R | 可能掩盖高N |
| D11否定词 | ↓↓ 极少直接否定（"没事""还好"） | ≠真实冲突处理风格；需与关系稳定后的文本对比 | 低估真实对抗性 |
| D10绝对化 | ↑ 积极向绝对化（"你最好""太完美了"） | ≠稳定认知风格；属热恋期常态 | 高估开放性低估A |
| D8人称 | ↑ "我们"增多，"我"适度 | ≠成熟的集体感；关系早期的"我们"是投射而非现实 | 高估关系成熟度 |
| D12因果链 | ↓ 减少（避免分析性破坏浪漫感） | 不作为O判断依据 | 低估认知复杂度 |
| D19假设性思维 | ↑ 探索性"如果"增多（"以后我们一起……"） | ≠人格中的开放性；早期关系常态幻想 | 高估O |
| D16承诺语言 | ↑↑ 高承诺密度，但具体度低（"永远""一辈子"） | ≠真实尽责性；早期关系承诺的预测力低 | 高估A |

**系数调整**：
- 面具厚度指数 × 1.3（但类型不同于工作：这是"粉红色面具"而非"职业面具"）
- 置信度 × 0.6
- 该阶段的分析**不可作为稳定依恋风格的依据**

**特殊归纳限制**：
- E（外向善良）/ R（人际关系）：早期亲密关系中的"热情温柔"可能是求偶策略，不作为稳定证据
- N（情绪性）：短期文本中情绪稳定很可能只是压抑，而非真稳定——如果文本中同时有≥2处情绪泄露痕迹（D5断裂句+D11否定+D20反刍），则N评分反而应上调
- **关系场景簇仅在有≥3个跨场景冲突端证据时才做依恋归纳**

**ND交互**：早期亲密关系是ND masking的高发区——ND4（社交脚本/masking）在此阶段可能极度激活（"装正常""装放松"）。如果ND4≥●中，E评分上调1-2分，关系场景中的"松弛"判断置信度降一级。

### ❹ 法庭/法律语境（Legal / Court / Police Statement）

**confound机制**：极端形式化环境，语言往往经过律师或专业人士起草/修改，或受到程序性约束（"如实回答""按格式陈述"）。**文本的作者不完全是被分析者**——这是LPP基本前提的边界案例。

**基线偏移**：

| D维度 | 法律语境偏移 | 校准规则 |
|-------|------------|---------|
| D1标点 | 完全规范（诉讼文书格式） | 完全不纳入评分 |
| D15正式度 | 极高且强制 | 不纳入任何判断 |
| D4句长 | 句式复杂但由程序决定 | 不可做O判断 |
| D9情绪词 | 趋零（法律语言要求客观） | 不纳入N/R判断 |
| D12因果链 | ↑↑ 强制性因果链（因为……所以……） | 不可做O判断 |
| D30情态动词 | 高度格式化（"应当""可以""不得"） | 不可做人格判断 |
| D38正当化 | 程序性正当化（法律论证要求） | ≠个人防御风格 |
| D35评价框架 | 法律用语的中性强制（"原告""被告""当事人"） | 不可做评价偏好判断 |
| D16承诺语言 | 承诺由法律条款而非个人意志决定 | 不纳入A评分 |

**系数调整**：
- **本层不做标准面具厚度指数计算**（面具模型不适用于法律文本）
- 置信度 × 0.3（极低）
- 仅分析文本中**程序之外的部分**（如当事人自述段落、情感性描述、非结构化的部分）
- 法律文本只做D1-D40的**基础扫描**，不做完整人格归纳
- 报告开头强制标注："样本来自法律语境，文本由程序规范和/或第三方起草影响，分析可靠性受限"

### ❺ 家庭传统场合语境（Family Events — 过年/家族聚会/祭祖）

**confound机制**：激活"家庭角色回归"——成年人回到父母家、家族聚会中，可能退回到青少年时期的语言模式（方言回归、语气词变化、顺从或对抗的家庭脚本）。这种**倒退本身掩盖了当前的人格发展水平**。confound方向与工作语境相反：不是变得更正式，而是变得更原始。

**基线偏移**：

| D维度 | 家庭传统场合偏移 | 校准规则 | 误读风险 |
|-------|----------------|---------|---------|
| D6语气词 | 可能↑↑（方言语气词回归） | 先做方言校正，再区分家庭角色语气vs自然语气 | 错判为高亲和 |
| D15正式度 | ↓↓ 显著低于日常社交往 | ≠真实权力距离态度；家庭角色激活 | 错判为低正式度人格 |
| D11否定词 | 模式切换（可能极端顺从或极端对抗） | 取决于家庭角色；不视为稳定否定模式 | 错判为高/低对抗 |
| D8人称 | 家庭角色影响（"咱家""我们李家"家族感↑） | ≠真实集体主义人格 | 错判为高R |
| D9情绪词 | 可能↑（亲情表达）或↓↓（压抑的家庭氛围） | 需结合家庭氛围推断 | 错判N |
| D5断裂句/修正 | 在冲突主题上可能↑ | 仅在重复模式时计入 | — |
| D14用词精确度 | 可能↓（口语化方言化） | 不纳入I判断 | 错判为低才干 |
| D32完结行为 | 家庭话题的收尾模式可能异于日常（被长辈打断/话题强制切换） | 不纳入A判断 | — |

**系数调整**：
- 面具厚度指数不适用（不是面具，是角色切换）
- 置信度 × 0.7
- 需与同一人在非家庭语境中的表达做对比后才有分析价值
- 单独的家庭文本（无对照）→ 仅做有限描述不做归纳

**ND交互**：家庭场合可能激活ND4（masking）的早期版本——"在家庭中扮演某个固定角色"。如果ND4≥●中，标注"家庭角色脚本可能独立于社交masking，需区分对待"。

### ❻ 匿名 vs 实名语境（Anonymous vs Real-name Expression）

**confound机制**：这一语境不适用"面具厚薄"模型——关键不是表达受抑制，而是**表达的目标发生了切换**。匿名时可能更真实，也可能是偏态释放（只发泄不建设）；实名时可能压抑负面，也可能有意识塑造个人品牌。核心问题不是"哪个更真实"，而是**同一个人的两套表达系统之间的差距本身携带心理信息**。

**基线偏移**：

| D维度 | 匿名语境典型变化 | 实名语境典型变化 | 校准规则 |
|-------|-----------------|-----------------|---------|
| D9情绪词 | ↑ 负面情绪释放 | ↓ 正面偏差（个人品牌） | 两套系统对比后取中位 |
| D10绝对化 | ↑↑ 极端表达增多 | ↓ 减少（怕被认出来） | 匿名端绝对化≤50%为表达风格，>50%为真实倾向 |
| D11否定词 | ↑ 直接否定增多 | ↓ 委婉化 | 取两端极值做上下界范围 |
| D6语气词 | ↑ 自然 | ↓ 控制 | 匿名端的语气词更接近自然基线 |
| D8人称 | ↓ "我"减少→匿名话题散 | ↑ "我"策略性增多 | 对比分析更有信息量 |
| D25网络话语标记 | ↑↑ 网络用语增多 | ↓ 减少 | 仅在匿名端做判断 |
| D35评价框架 | 对他人的评价两极分化 | 中性化 | 取两者倾向方向做中心估计 |

**系数调整**：
- 不适于面具厚度指数（不是同一层面概念）
- 如果同时拥有同一人的匿名和实名样本 → **极有价值的分析素材**
  - 两套系统的差异幅度越大 → 反映越强的**语境调制能力**（本身是一个独立心理特征）
  - 两套系统在R/A/N维度上的差异方向 → 反映"对社会评判的在意程度"
  - 差异主要集中在D9/D10/D11/D35 → 反映情绪/攻击性的社会抑制水平
  - 差异集中在D8/D15/D30（人称/正式度/情态）→ 反映社交角色切换能力
- 仅有一个端（只有匿名或只有实名）→ 置信度×0.6，标注"仅分析了表达系统的一端"

**报告呈现规则**：
- 任何匿名文本的分析必须在报告开头标注"样本来自匿名语境，可能包含偏态释放"
- 任何实名文本标注"样本来自实名语境，可能受个人品牌建设影响"
- 双端样本 → 在人格归纳中以两端的合成评分 + 差异幅度作为额外维度呈现

### ❼ 创伤后/危机中表达（Post-trauma / Crisis Expression）

**confound机制**：情绪强度主导全部语言模式——D9/D20/D5/D11被全面拉升。这不是面具问题（没有主动隐藏），而是**状态压倒特质**——类似于测血压时刚跑完步。文本反映的是当前的情绪状态，而非稳定的心理结构。

**基线偏移**：

| D维度 | 危机语境偏移 | 校准规则 | 误读风险 |
|-------|------------|---------|---------|
| D9情绪词 | ↑↑ 情绪词密度大幅升高 | 负面情绪词密度×0.5校正后进入评分 | 错判为极高N |
| D5断裂句 | ↑↑ 自我修正、未完句增多 | 标记为"危机下句法断裂"，×0.3权重 | 错判为认知混乱/低I |
| D20反刍循环 | ↑↑ 同一主题反复回访 | ≠稳定反刍倾向；可能为急性应激 | 错判为反刍型人格 |
| D11否定词 | ↑↑ 防御性否定增多（"不""没有""别"） | ×0.5校正 | 错判为高对抗性 |
| D10绝对化 | ↑↑"永远不""完全没救" | 标记为危机语言特征，×0.3 | 错判为极端化认知 |
| D12因果链 | 单因单果增多（简化归因） | ≠低认知复杂度；应激下简化是常态 | 错判为低O |
| D13话题聚焦 | ↑ 高度聚焦于危机事件 | ≠高尽责；危机性聚焦 | 错判为高A |
| D37动作-状态动词 | ↑ 状态动词（"好难受""撑不下去了"）剧增 | 不可用动作-状态比做人格判断 | — |

**系数调整**：
- 不做面具厚度指数（与面具概念无关）
- 置信度 × 0.4（状态掩盖特质，信度极低）
- 关键判断规则：**危机文本中出现的任何负面信号，不能单独作为人格归纳证据**

**特殊归纳限制**：
- N（情绪性/大七）/ N（神经质/大五）：危机文本中该维度必然虚高——任何危机文本的高N都不得轻信。低N判断（情绪稳定）仅在文本中明确有正向情绪调节和认知重构证据时才可做
- O（开放性）：危机中的认知窄化是应激反应，≠低开放性
- I（才干）/ C（尽责性）：危机中的执行功能下降是状态性的，≠低I/A
- 人格归纳至少延后到危机缓解后的文本进行；仅有危机文本时标注"样本来自危机期，人格归纳不可靠"

**ND交互**：危机可能暴露平时被掩盖的ND特征——ND1（感官敏感）在高压下增强、ND3（注意力切换）在资源耗竭时加重。如果危机文本中ND信号强度明显高于非危机文本中同一人的ND信号，以非危机文本为准，标注"危机状态可能放大了ND特征"。

**情绪轨迹整合**（本语境适用情绪轨迹分析的加强版）：
- 危机文本必须做完整的情绪轨迹分析
- 如果情绪轨迹呈**上升型**（从负面转向正面），说明有自我调节能力——即使当前D9负面词高，人格评分中的N可适度下调
- 如果情绪轨迹呈**下降型**（持续恶化且无反弹），维持在原有低置信度

### ❽ 治疗/咨询语境（Therapy / Counseling）

**confound机制**：来访者的表达可能受以下因素系统偏倚——（1）"好病人"效应：希望被喜欢/被认为有进步，呈现更积极/更配合的版本；（2）过度病理化：想获得关注或诊断，放大症状描述；（3）疗愈视角污染：经过心理咨询概念框架过滤后的自我描述（"我的原生家庭……""我的内在小孩……"）——**语言本身已被理论框架重塑**；（4）与咨询师的权力关系：依赖、移情、对抗等因素影响表达方向。confound 方向不确定（可能向上偏也可能向下偏），但**偏倚幅度可以很大**。

**基线偏移**：

| D维度 | 咨询语境偏移 | 校准规则 |
|-------|------------|---------|
| D9情绪词 | 可能↑（鼓励表达感受）或↑↑（过度病理化） | 需判断文本中是否有"理论框架污染"（使用专业术语描述自我） |
| D34引用策略 | ↑↑ 引用咨询师/心理学概念（"我的咨询师说……""书上说……"） | 被理论框架重塑的自我描述≠自然自我描述 |
| D20反刍循环 | ↑↑ 对同一主题系统性回顾 | 咨询语境中反刍是结构性安排，≠稳定反刍倾向 |
| D12因果链 | ↑↑ 因果归因密集（"因为……所以我……"） | 被咨询过程引导的分析性表达，不可做O判断 |
| D38正当化密度 | ↑ 堆叠（咨询鼓励"展开说说"） | ≠防御性 |
| D35评价框架 | 可能用心理学标签评判自己与他人 | 标签化评价（"自恋""控制欲""回避型"）不做人格判断 |
| D30情态动词 | 认识情态↑（"我可能……""我总觉得……"） | — |
| D36认知谦逊 | 高但可能由咨询框架塑造（"我意识到……""换个角度看……"） | ≠真实的认知谦逊水平 |

**系数调整**：
- 不做面具厚度指数（与面具概念无关，是框架污染问题）
- 置信度 × 0.5

**核心区分规则**——区分两种使用模式：

| 模式 | 识别信号 | 处理方法 |
|------|---------|---------|
| **模式A：咨询过程的自然言语**（咨询中的对话记录） | 对话格式、有咨询师回应、话题开放 | 可用标准流程但标注框架污染可能；D34引用密度≥3处的维度置信度降一级 |
| **模式B：来访者的自述/日记**（咨询之外的文本或咨询师要求写的材料） | 独白格式、主题聚焦于自身 | 比A模式更接近自然自我；但仍需注意"理论框架污染" |

**理论框架污染检测**：
- 统计文本中出现的心理学/治疗术语密度（"原型""投射""防御""边界""原生家庭""内在小孩""triggered""创伤反应"等）
- 术语密度 ≥ 每300字1个 → 标注"受理论框架中度污染"
- 术语密度 ≥ 每100字1个 → 标注"受理论框架重度污染"，剔除该部分→剩余文本做标准分析
- 受重度污染的段落独立标注，不作为人格归纳依据

**ND交互**：充分注意**ND掩盖**——ND来访者可能在咨询中发展出高度的分析性语言来补偿社交困难，文本中的高级心理分析和自我洞察可能反映的是认知补偿努力（ND5↑），而非真实的高O或高I。如果ND5≥●中且D34引用密度同时高，O/I置信度各降一级。

## ⚠️ 时间跨度校准层（Temporal Drift Confound）

**核心问题**：人格不是永恒不变的。当分析样本由跨时间碎片拼凑而成时，碎片间的语言差异可能部分来自**人格的真实变化**（development/drift），而非测量噪声或情境调制。如果不做时间跨度校准，可能会把真实变化误判为"高情境敏感性"或"低特质稳定性"。

### 人格稳定性的已知基线

| 年龄段 | 人格稳定性水平 | 对分析的启示 |
|-------|--------------|------------|
| **青少年（≤22岁）** | 低-中，仍在人格塑造期 | 跨2年以上碎片间的差异很可能包含真实变化 |
| **青年（23-30岁）** | 中，过渡期 | 跨3年以上需考虑可能性，着重标记 |
| **中年（31-50岁）** | 高，人格趋于稳定 | 跨5年以上才需考虑，但关键生命事件（离婚/失业/丧亲等）可触发变化 |
| **中老年（50+）** | 高-中，可能回归或软化 | 跨5年以上+重大事件才标记 |

**关键事实**：在无重大生命事件的情况下，中年人格的5年retest相关性约为 r=0.7-0.8（各维度不同：N和E相对更稳定，O随年龄变化更大）。这意味着即使"稳定"，也非完全不变。

### 时间跨度偏移的检测方法

在Input Preparation阶段，除了已有拼凑检测（时间碎片标记/对象碎片标记/情景碎片标记），增加**时间跨度评估**：

```
时间跨度检测清单：
□ 文本中是否出现不同时期的时间参照？（"去年""三年前""刚毕业时""最近"等）
□ 已知的来源时间范围（如收集自2019-2025年的帖子）
□ 文本中是否包含年龄/阶段变化标记？（"那时候我还小""工作以后""有了孩子之后"）
□ 是否包含重大生命事件的时间参照？（"离婚前""他走后""跳槽之后"）

判定：
  无时间参照 / 时间参照集中在≤6个月 → 时间跨度可忽略
  时间参照跨度1-3年 → 需注意（轻度校准）
  时间参照跨度3-5年 → 需校准（中度校准）
  时间参照跨度5年+ → 强制校准（强校准）
  已知来源时间范围 → 直接按实际跨度判断
```

### 时间跨度校准规则

#### 跨度过大时的处理流程

当检测到时间跨度 ≥3年时（或跨度的年龄段处于人格不稳定期），按以下规则处理：

```
Step 1：按时间碎片切分 → 对每个碎片独立做Step 1扫描
Step 2：各碎片独立做Step 2场景定位
Step 3: 
  对各碎片独立做人格归纳
  ↓
  判定变化类型：
  ├─ 稳定型：所有碎片评分指向同一半区（差异≤2分）→ 用碎片加权均值做评分
  ├─ 趋势型：评分沿时间顺序单向变化（T1<T2<T3或T1>T2>T3）→ 报告时间变化轨迹而非单一评分
  ├─ 波动型：评分高低交替，无方向 → 用均值标注波动范围
  └─ 事件触发型：在某时间点骤变，前后稳定 → 锚定事件前后的不同人格状态
```

#### 变化类型的具体处理

**趋势型变化**——人格在时间轴上单向移动：

| 变化方向 | 可能的含义 | 报告呈现 |
|---------|-----------|---------|
| N（情绪性）↓ | 成熟/心理成长/情绪调节能力增强 | 标注"跨时间呈现N下降趋势，符合心理成熟轨迹" |
| E（外向善良）↓ | 可能反映社会性消耗或价值观内收 | 标注"跨时间E下降信号" |
| A（行事风格）↑ | 尽责性随年龄增长是跨文化稳定规律 | 标注"符合尽责性随年龄增长的一般规律" |
| O（开放性）↓ | 中年期开放性自然下降 | 标注"O变化可能与年龄段相关" |
| R（人际关系）↑ | 反映包容度的提升（如为人父母后） | 标注"跨时间R上升趋势" |
| T（处世态度）→ | 处世态度在大七中相对稳定 | 标注"T跨时间稳定" |

**处理规则**：
- 趋势型 → 在报告中以变化轨迹呈现，不输出单一评分
- 如果趋势方向符合该年龄段的已知发展规律（如A↑、O↓、N↓）→ 标注"该变化方向与年龄发展一致"
- 如果趋势方向与年龄段预期相反 → 标注"变化方向与一般年龄趋势不一致，可能反映个体特异性经验"

**波动型变化**——评分高低交替无方向：
- ≤1分波动 → 视为测量噪声，取均值
- 2-3分波动 → 标注"人格维度呈现跨时间波动"，以区间而非单值呈现（如"N 5-7/10"）
- ≥4分波动 → 检查碎片的语境或ND特征是否有系统性差异（如工作碎片vs个人碎片交替）

**事件触发型变化**——在某时间点出现骤变：
- 识别触发事件（如有时间参照提示）
- 在报告中呈现"事件前后的人格差异"
- 综合评分以事件后的最新状态为主（假设当前人格状态更接近现在），事件前的状态作为"历史背景"

### 时间跨度 × 语境 type 交互校准

当时间跨度校准与语境校准同时生效时：

| 时间跨度 | 语境类型 | 交互规则 |
|---------|---------|---------|
| ≥3年 | 工作语境/公开展示社媒 | 语境可能mask了人格变化——同语境下的变化可能被职业角色的一致性压制。优先用碎片中的个人语境部分做判断 |
| ≥3年 | 治疗/咨询 | 咨询过程中的变化可能本身就是治疗目标——应当将变化视为分析的核心发现而非confound |
| ≥3年无明确时间标记 | — | ⚠️ 无法区分"变化是时间引起的"还是"变化是情境引起的"。置信度额外×0.8 |
| 青少年（≤22岁）且跨≥2年 | 任何语境 | 人格变化概率高。所有评分以区间呈现（"E 5-8/10"），不做单值 | 

### 时间跨度对置信度的影响

| 时间跨度 | 年龄段 | 置信度附加衰减 | 特殊处理 |
|---------|-------|--------------|---------|
| 无可检测时间跨度 | 任何 | ×1.0 | 标准流程 |
| ≤1年（有明确离散时间点） | 任何 | ×0.9 | 标注"跨[时长]碎片" |
| 1-3年 | 青少年 | ×0.7 | 以区间呈现评分 |
| 1-3年 | 青年+ | ×0.85 | 检查趋势方向 |
| 3-5年 | 青少年 | ×0.5 | 强制按变化类型处理 |
| 3-5年 | 青年+ | ×0.7 | 识别变化类型 |
| 5年+ | 任何 | ×0.4-0.5 | 按趋势型/事件触发型处理，不输出单一评分 |

### 与现有"拼凑样本"规则的融合

现有的拼凑样本规则（系统提示层）主要处理**碎片来源的差异**（时间/对象/情景）。时间跨度校准在此基础上追加：

**融合处理流程**：

```
Input Preparation：
第一步：常规拼凑检测 → 识别并标记碎片（时间/对象/情景）
第二步：时间跨度评估 → 评估碎片间的时间跨度
第三步：如果时间跨度 ≥3年（或跨人格不稳定期）：
   ├─ 不做常规的"拼凑均值+方差衰减"处理
   └─ 改为执行"变化类型判定"（稳定/趋势/波动/事件触发）
如果时间跨度 <3年：
   └─ 使用常规拼凑样本处理规则
```

**碎片的离散度标注规则**（在时间跨度≥3年时修改）：

现有规则：
```
碎片间方差 → 置信度衰减
```

修改为：
```
碎片间方差 → 先判定变化类型：
  ├─ 趋势型：方差不衰减置信度，改为报告变化轨迹
  ├─ 波动型：方差衰减置信度（与原规则一致）
  ├─ 事件触发型：锚定事件前后分别标注
  └─ 稳定型：方差不衰减（差异≤2分视为测量噪声）
```

### 报告中的呈现规则

当时间跨度校准激活时：

1. **开头标注**："样本跨[时长]年（[起始年份]至[结束年份]），时间跨度分析表明人格[变化类型]，以下评分以[最新状态/变化区间/变化轨迹]呈现。"
2. **评分条格式调整**：
   - 稳定型 → 正常评分条
   - 趋势型 → 箭头标注各时间点位置：`■ E 外向善良 5→6→8 ████████░░ [T1-T3趋势:上升]`
   - 波动型 → 区间标注：`▲ N 情绪性   ██▒▒▒▒████░░ [3-7波动]`
   - 事件触发型 → 锚点标注：`■ A 行事风格 ████████░░ [事件后] (事件前:4)`
3. **各碎片评分展示**：在各维度评分条下方以脚注形式展示碎片原始评分
4. **依恋归纳**：仅当时间跨度≤2年或个人语境碎片≥2个且指向同一类型时才做归纳；否则标注"时间跨度长+关系场景稀疏，不做依恋判断"

### 年龄/阶段变化标记检测（补充D观察）

在D1-D40扫描中补充以下观察条目（不增加新D编号，作为时间跨度检测的辅助线索）：

| 语言标记 | 可能反映的阶段性变化 |
|---------|-------------------|
| "以前我……但现在……" / "曾经……后来……" | 明确的自我认知变化叙事 |
| "年轻的时候……现在……" | 年龄成熟叙事 |
| "经历了XX之后……" / "从那以后……" | 事件触发型变化 |
| 对同一主题在不同碎片中有不同的评价框架(D35) | 价值观/处世态度的变化可能 |
| 人称模式(D8)在早期vs晚期碎片中系统性变化 | 社会角色感的变化 |
| 情态动词分布(D30)在早期vs晚期碎片中的变化 | 确定性/认知成熟度的变化 |
| 绝对化词汇(D10)的跨碎片密度差异 | 认知弹性的发展变化 |

## ⚠️ AI生成/辅助文本校准层（AI-Generated / AI-Augmented Text）

**confound机制**：当文本经过AI（如ChatGPT、Claude等）生成或润色，语言模式被系统性地中性化/平均化——句式趋于规整、语气词被删除、绝对化被软化、情绪表达被调节、人称模式被"更得体"地重构。**文本的语言形式模式反映的是模型偏好而非作者特征**，这是LPP基本前提（"语言具有稳定可预测的个人模式"）的致命confound。

**检测方法**：

| 检测线索 | 判断依据 |
|---------|---------|
| 句式异常规整 | 全文句长分布均匀，从句结构模式单一，无自然口语的长度变化 |
| 无个人化碎片 | 无D5断裂句/自我修正、无D25填充词、无D6自然语气词 |
| 标点完美但呆板 | D1全角规范+无个性标点偏好（无波浪号/非常规省略号） |
| 段落结构模式化 | 每段长度接近、开头结尾结构重复（如总是先总结后展开） |
| 词汇搭配共性化 | 特定AI高频词（"然而""值得注意的是""从这个角度""本质上"）密集出现 |
| 中英混杂的AI腔 | 翻译痕迹明显的句式（长定语从句、被动语态、形式主语） |
| 作者主动声明 | "用AI写的""ChatGPT润色过""AI帮我改的" |

**分层处理规则**：

| 检测结果 | 处理方式 |
|---------|---------|
| **全篇AI生成** | **不分析**。在报告开头标注"该文本经检测为AI生成，不具备个人语言特征，LPP分析不适用" |
| **AI辅助润色**（大段改写） | 仅分析润色前的核心表达部分。润色痕迹明显的段落标注"疑似AI润色影响"。整体置信度×0.3 |
| **AI局部调整**（个别词句） | 对疑似被AI改写的词句周围段落做降权处理（权重×0.5）。其余部分可正常分析，标注"部分内容经AI调整" |
| **疑似但不确定** | 标注"文本存在AI生成痕迹的可能"，置信度×0.6，纳入"LLM验证层"做AI文本概率评估 |
| **无检测痕迹** | 标准流程 |

**特殊限制**：
- AI生成的"第一人称"文本（如"写一封我向客户道歉的邮件"）中的人称代词(D8)不反映任何人格特征
- AI润色后的文本中D1标点、D15正式度、D21表达节奏不可纳入任何人格判断
- 如果同一人的AI文本和非AI文本同时存在，以非AI文本为准做完整分析，AI文本仅做参考对比

## ⚠️ 非母语表达校准层（L2 / Heritage Speaker Confound）

**confound机制**：非母语者的语言模式受限于语言能力而非认知风格——词汇库受限、句式简化、情态表达不精确、语气词使用异常。这些**语言学上的限制**容易被误读为**人格特征**（"句式简单=认知简单""语气词少=低亲和"）。同样问题适用于传承语者（海外华人，其中文发展在某一阶段停滞）。

**基线偏移**：

| D维度 | L2/传承语者偏移 | 可能误读方向 | 校准规则 |
|-------|----------------|-------------|---------|
| D4句长/从句 | ↓↓ 简化和碎片化 | 低O/低I | L2中句长简单≠低认知复杂度。标注"语言能力限制" |
| D14用词精确度 | ↓↓ 同词重复、模糊词汇多 | 低I/低A | 词汇广度受限于语言习得水平。不纳入I/A评分 |
| D6语气词 | ↓↓ 非母语者自然低频或异常高频（非自然分布） | 低E/低亲和 | 不纳入E/R评分。仅当语气词使用自然且高频率时才计入 |
| D12因果链 | ↓ 因果连接词简单（"因为""所以"为主） | 低O | 非母语者先掌握基础因果连词。复杂因果链不反映认知限制 |
| D9情绪词 | ↓ 密度低或表达不自然 | 低N | 情绪词汇广度受限。不基于密度做N判断 |
| D11否定词 | 过于直接或过于委婉（语用迁移） | 对抗性/不自信 | 语用规则跨文化差异。不纳入对抗性判断 |
| D10绝对化 | 可能↑（非母语者两极化的表达策略） | 极端的认知风格 | 不一定反映真实认知风格。≥5处才标记 |
| D30情态动词 | 精确度下降（"可能""也许""应该"混用） | 不确定感 | ≠真实的不确定容忍度。不纳入N/O判断 |
| D21表达节奏 | 异常（母语节奏迁移） | 情绪/表达方式 | 标注"母语节奏影响"，不纳入人格判断 |

**系数调整**：
- 语言水平评估：**高级**（接近母语）→ ×0.9，标注"可能存在L1迁移影响"
- **中级** → ×0.6，频繁标注语言能力confound
- **初级** → 不分析完整人格，仅做有限观察描述
- **传承语者** → ×0.7（语言形式系统部分特质但仍受限）

**ND交互**：ND分析在非母语文本中需格外谨慎——ND4（masking标记）在L2中会异常升高（非母语者始终处于"在装"状态），ND5（认知补偿）在L2中可能激活但反映的是语言学习策略而非认知风格。L2文本中ND4↑/ND5↑的confound幅度降半。

**母语迁移特征排除**：
- 日韩母语者→中文的SOV语序残留、主语省略习惯、敬语迁移
- 英语母语者→中文中过多的主语、被动语态、定从句
- 东南亚华裔→中文词汇的在地化使用
- 西方语言母语者→标点习惯迁移（英文省略号间距、引号使用等）
- 这些特征排除后剩余的观察才进入Step 2/3

## ⚠️ 翻译文本校准层（Translated Text）

**confound机制**：翻译文本**最为致命**——文本的语言形式模式来自译者而非被分析者。原作者的风格（句式长短、人称习惯、语气词等）几乎全部滤失，译者的风格取而代之。这是LPP框架中最严重的confound——大多数D维度的观察在翻译文本中完全失效。

**处理规则**：

| 翻译类型 | 处理方式 | 置信度 |
|---------|---------|--------|
| **专业翻译**（出版级，经过编辑校对） | **不分析**。标注"该文本为翻译文本，不满足LPP分析前提" | ×0 |
| **机器翻译**（DeepL/Google Translate等） | **不分析**。标注"机器翻译无法保留个人语言特征" | ×0 |
| **非专业人工翻译**（个人口译转写、粗略翻译） | **不分析**。仅做内容层面的主题分析，不做语言形式分析 | ×0 |
| **双语对照**（原文+翻译同时存在） | **以原文为准**做标准分析。翻译文本仅做参考 | ×1.0（原文） |
| **自译**（作者本人翻译自己的作品） | 作者可能保留部分个人模式。但风险极高——仍以原文为主分析 | ×0.3 |

**可保留的分析**（即使在翻译文本中）：
- D13话题聚焦度（话题范围在不同翻译中相对稳定）
- D17归因风格（归因的对象和方向可能在翻译中保留）
- D35评价框架（褒贬偏好在大意层面可能保留）
- 上述三个维度的置信度仍需要降为▲弱信号

**报告标注**：
- 任何程度的翻译文本在报告中以⚠️开头标注翻译confound
- 原文语言非中文/英文时需注明原文语言

## ⚠️ 群体/代言表达校准层（Group / Representative Speech）

**confound机制**：当作者代表某个群体发言（官方声明、团体发言、组织公开信、会议纪要、群公告），人称模式、评价框架、情态系统全面切换至"代表模式"——"我"变成"我们"或隐去，个人评价让位于组织立场，情态动词变为制度化语言。文本反映的是**发言人角色**而非个人。

**检测清单**：
```
□ 文本是否明确以群体名义发表？（"我方""本公司""组委会"等）
□ 是否使用机构署名或代表身份标识？
□ 人称是否全程为"我们"而无"我"？
□ 评价是否明显遵循某一立场而非个人判断？
□ 情态是否制度化（"应""须""经研究决定"）而非个人表达？
```

**校准规则**：

| 文本类型 | 处理方式 |
|---------|---------|
| **正式官方声明**（机构/公司/政府） | 不分析人格。仅做主题和修辞策略分析 |
| **组织内代表发言**（会议发言、项目汇报） | 区分"被迫代表"（读PPT/背稿）vs"主动表达"。后者中未经准备的部分（Q&A、自由讨论）可部分分析 |
| **群公告/群管理言论** | 不纳入D8人称判断、"我们"的使用不可做任何集体主义推断 |
| **集体创作文本**（多人执笔的联合声明） | 不分析。多人执笔导致语言模式混合 |
| **个人以社群身份写作**（如"作为一个XX主义者"） | 标注"可能受群体身份影响"。D35评价框架中与该群体立场一致的判断不纳入R评分 |
| **AI辅助的团队沟通**（AI写的周报/项目总结） | 叠加AI生成confound → 不分析 |

**特殊限制**：
- 代言文本中的D8人称模式不纳入任何R/I评分
- 代言文本中的D35评价框架——与组织立场一致的评价不纳入R评分，与组织立场相悖的评价（如有）则是高价值的真实信号
- 代言文本中的D30情态——制度化情态（"应""须""经研究"）不纳入人格判断
- 置信度 × 0.5（即使是最"个人化"的代表文本）

## ⚠️ 平台/媒介文化校准层（Platform / Medium Culture）

**confound机制**：每个社交平台/媒介介质形成了独特的**表达规范和文体期待**——Zhihu推崇长篇论证体、Xiaohongshu偏好体验式短叙事、Weibo限制字数催生情绪浓缩、Bilibili向评论区扩散的弹幕文化、WeChat私聊的极度碎片化。同一人在不同平台上会有系统性的表达差异，**这些差异不是人格的跨情境变化，而是媒介语境的规约效果**。

**各平台基线特征**：

| 平台/媒介 | 典型语言特征 | 易误读方向 | 校准规则 |
|----------|------------|-----------|---------|
| **知乎（Zhihu）** | 长段落、因果链密集(D12↑)、引用策略发达(D34↑)、正式度高(D15↑) | 高O/高I | 文本的"分析性"是平台文体要求，≠个人特征。正式度降1-2分校正 |
| **小红书（Xiaohongshu）** | 短段落+emoji多(D3↓)、情绪词密集(D9↑)、语气词多(D6↑)、绝对化多(D10↑) | 高E/高N/低A | 平台鼓励情感化+夸张表达。情绪词密度自然偏高，D10绝对化降权 |
| **微博（Weibo）** | 字数限制→断裂表达(D5↑)、绝对化(D10↑)、情绪浓缩(D9↑)、话题跳跃(D13↓) | 高N/低A/高冲动 | 字数限制带来的表达压缩。自然D5/D10升高，降为▲弱信号 |
| **微信聊天（WeChat）** | 碎片化(D3↓)、语气词丰富(D6↑)、填充词多(D25↑)、随性断句(D2↓规范) | 低A/低I | 即时聊天的碎片化是平台常态。D2/D3模式的基线参照文体校准表 |
| **B站评论区（Bilibili）** | 网络梗密集(D25b↑)、碎片互动(D5↑)、语气活跃(D6↑)、调侃多(D26可能↑) | 低I/低A | 社区文化偏好轻松互动风格。D26↑做社交调侃区分的严格标记 |
| **豆瓣（Douban）** | 长评分析性强(D12↑)、情绪表达细腻(D9↑)、自我暴露多(D8"I"↑) | 高O/高N | 社区文化鼓励自剖。I-talk升高是社区规范而非个人 |

**混合平台样本处理**：
- 当样本来自单一平台 → 应用平台基线校正后进入标准流程
- 当样本来自多个平台 → 碎片间差异优先归因为平台效应而非情境调制，标注"平台间差异可能反映媒介规约差异而非情境敏感性"
- 平台校正后的观察模式中仍跨平台一致的，才是稳定的个人模式

**媒介效应附加**（同一平台内）：
- 手机输入 → D1标点不规范（效率因素）、D3段落极短（屏幕限制）。不纳入A评分
- 电脑输入 → D1可能更规范、D3段落更长。基线正常
- 语音转文字 → D5断裂句↑、D25填充词保留、D9更自然。语音文本中的D5/D25接近口语基线

## ⚠️ 性别社会化语言基线（Gender Socialization Baseline）

**confound机制**：在中文社会化环境中，性别角色规范系统性影响某些D维度的基线——女性被社会化倾向更温和、更关系导向的表达（更多语气词、更多情感词、更多间接否定），男性被社会化倾向更直接、更抑制情感的表达（更少语气词、更少情感词、更多直接否定）。**这些差异反映社会化的语言习得，不必然反映人格特质差异**。LPP若不校正，可能对同一人格维度的评分产生跨性别的系统性偏差。

**基线差异表**：

| D维度 | 社会化典型差异 | 可能误读 | 校准规则 |
|-------|--------------|---------|---------|
| D6语气词 | 女性基线高于男性（中文实证稳定差异） | 女性倾向"高E/高R"，男性倾向"低E/低R" | 以群体中位值为基准做偏移校正。女性D6↓在基线之上才是有效信号；男性D6↑在基线之上才是有效信号 |
| D9情绪词 | 女性基线高于男性（情绪表达的社会许可差异） | 女性倾向"高N"，男性倾向"低N" | 情绪词密度超出同性别基线1SD以上才进入Step 3。±1SD以内视为基线噪声 |
| D11否定词 | 女性倾向委婉否定（"可能不太行"），男性倾向直接否定（"不行"） | 女性"低对抗"，男性"高对抗" | 当D11需要纳入对抗性判断时，需以自身基线（该人在真实冲突场景中的表达）为参考而非群体基线 |
| D8人称 | 女性"I-talk"可能略高于男性 | 女性"高自我关注" | I-talk跨性别差异效应量小（d≈0.2），仅在极端差异（>2SD）时标记 |
| D10绝对化 | 男性可能略高于女性 | 男性"高极端化" | 效应量小，不作独立校正。仅在工作或早期亲密关系等对D10影响更大的语境中优先做语境校正 |
| D35评价框架 | 女性倾向更温和的评价语言 | 女性"高R" | 社会化对评价框架的影响不稳定。不作独立批量校正，由分析师在Step 3归因时标注"不能排除性别社会化影响"的可能 |

**重要原则**：
1. **个体差异远大于群体差异**——性别基线校正是统计层面的偏倚校正，不影响个体内的特质判断
2. **只校正，不诊断**——校正后的差异可能反映真实的人格差异，也可能反映其他社会化因素
3. **跨性别样本的对比不推荐**——不同性别的基线差异使得跨性别直接比较不可靠
4. **非二元/酷儿表达者的特殊位置**——上述社会化基线可能不完全适用，标注"性别社会化表达基线不确定"
5. **时效性**——性别社会化正在变化中，上述基线在校准2020s后期的文本时权重降为0.7

**使用规则**：
- 性别基线校正仅在该D维度的**极端判断**（1-3或8-10）中应用
- 中位区间的判断不做校正（性别基线差异在该区间内不具区分力）
- 校正仅标注在报告第二部分，不出现在第一部分的叙事中
- 当性别基线校正与语境校准方向相反时，以语境校准为准（语境效应 > 性别效应）

## ⚠️ 权力不对称关系校准层（Power-Asymmetric Relationships）

**confound机制**：在工作语境之外，权力不对称关系同样普遍存在并系统性压制表达——师生关系中学生自动收敛异议、医患关系中患者倾向顺从、咨询关系中来访者配合导师、学术关系中研究生对导师的措辞谨慎。这些**话语权力关系**不同于"职业角色"（不是因为有工作职能），而是因为关系中一方的评价权/资源分配权高于另一方。即使关系本身不是职业性的，权力压制的语言效应仍然存在。

**常见权力不对称关系**：

| 关系类型 | 低位者的典型表达偏移 | 校准规则 |
|---------|-------------------|---------|
| **师生/师徒** | D6↓语气词抑制、D11↓委婉否定、D15↑高度正式、D35↑对老师评价正面偏移 | 关系中出现的正式语言和正面评价偏移→不纳入E/R/A判断。仅在关系之外的自由表达中出现的相关模式才有效 |
| **医患** | D11↓↓几乎不否定医生、D9↑情绪表达可能增多（倾诉效应）或↓↓（临床压抑） | D11在医患语境中独立于人格。D9在医患语境中的方向需结合具体关系判断 |
| **学术指导关系**（导师-研究生） | D15↑↑极端正式（学术邮件风格）、D8"我"退缩到"我们"甚至被动语态 | 学术沟通中的"我们"≠集体主义。极端正式的表达=权力距离标记≠人格 |
| **客户-服务者**（独立于具体职业） | D6↑刻意友善语气、D11↓↓完全回避否定、D15受行业规范 | 服务关系中的"友善"是角色要求。校准规则参考"工作语境"中的对应角色类型 |
| **代际权力**（子女对父母的经济/决定依赖） | D8人称切换（回避直接挑战）、D11委婉化或完全抑制、D10绝对化减少 | 对父母的"礼貌"反映代际权力结构，≠人格的权力距离感知。校准参考"家庭传统场合" |
| **其他不对称**（宿管/管理员/社区领导/房东等） | 偏向正式礼貌+回避否定+人格隐藏 | 按不对称程度：高不对称→参考"面试语境"的极端处理；低不对称→标注confound不执行批量校准 |

**通用校准规则**：
- 权力不对称关系中出现的任何"礼貌/正式/顺从"语言模式，**不作为人格归纳中的E/R/A低分证据**
- 权力不对称关系中出现的对权力方的正面评价(D35)，不作R高分证据
- 权力不对称关系中有且仅有的**冲突/拒绝/负面评价信号**具有极高人格信息量（权力压力下的真实表达）
- 置信度：权力关系主导的文本×0.6；关系中的非权力部分×0.8

**与工作语境的关系**：
- 工作语境中的上下级关系已经由"工作语境校准层"处理
- 本条覆盖**非工作语境中的权力关系**（师生、医患、代际、学术等）
- 同时存在于工作语境的权力关系 → 以工作语境校准为主，本层作为补充

## ⚠️ 临床/亚临床状态 confound 校准层（Clinical / Subclinical State）

**confound机制**：未治疗或未识别的临床/亚临床心理状态（区别于急性危机）可能系统性改变语言模式，而个体可能不意识到或未报告。这些状态不是"面具"（主动隐藏），也不是"危机"（急性发作），而是长期存在的状态性偏移——**它们使语言模式系统偏离人格基线但不被识别为confound**。

**常见临床/亚临床状态的表达偏移**：

| 状态类型 | 典型语言偏移 | 可能误读方向 | 校准规则 |
|---------|------------|-------------|---------|
| **抑郁症**（未治疗/亚临床） | D9情绪词↓或空洞正性、D8"I-talk"↑↑、D4句长↓（认知迟缓）、D12因果链简化、"不"型否定增多(D11↑) | 高N+低I+低A | 区分"抑郁状态语言"与"稳定人格"。I-talk↑+情绪词↓同时出现→高度怀疑抑郁状态影响。N评分在该组合下至少降1分 |
| **广泛性焦虑** | D19"如果"大量（焦虑性"如果"）、D24不确定性回避↑、D10绝对化→正向控制型（"必须确保"）、D5断裂句↑ | 高N+低T(处世态度) | 焦虑的"如果"以灾难性为标志（"如果出事怎么办"），区别于探索性"如果"。焦虑性如果≥3处→N降1分 |
| **SSRI类药物影响** | D9情绪词密度下降（情感钝化）、D9正面/负面同时降低、D20反刍减少、D6可能↓ | 低N | 药物导致的情绪扁平化≠人格的情绪稳定性。N评分在确认用药信息后上调1分 |
| **述情障碍/情感淡漠**（非ND源） | D9极低（情绪词汇贫乏）、D37状态动词↓（情绪感受描述少）、D8"I-talk"低（内省少） | 低N+低E | 区分临床述情障碍与ND认知风格。仅当D9极低且无ND1-ND6任何标记激活时才标记述情障碍可能性 |
| **ADHD未诊断/未治疗** | 与ND3（注意力切换）表现重叠。D3段落碎片化↑、D13话题跳跃↑、D5↑断裂句 | 低A+低I | ADHD的未诊断状态confound已在ND层中部分覆盖。本层标记：当ND3≥●中且无自我标签时，A评分不降（执行功能≠人格），标注"可能存在未诊断的ADHD影响" |
| **强迫相关** | D10绝对化↑（规则类）、D30道义情态↑↑（"必须""应该"）、D2句号极度规范(D1) | 低O+高A | 规则语言在OCD语境中反映的是症状而非人格的"严谨"。O评分在该模式下不下调，A评分上调的置信度衰减 |

**通用检测策略**：
- 不能仅从语言模式做诊断（LPP不诊断）
- 仅标记"语言模式与[状态类型]的表达特征一致"，标注"不能排除临床/亚临床状态的表达影响"
- 同一人的临床状态文本 vs 稳定期文本同时存在时，以稳定期为准
- 仅有状态期文本 → N维度置信度自动降为▲弱，其他受影响的维度同样降一级

**伦理要求**：
- 报告中不使用"抑郁症""焦虑症"等诊断标签，改写为"情绪困扰状态""持续焦虑状态"等描述性语言
- 标注"本观察不能替代专业心理评估"
- 临床状态的confound标记仅在分析师有充分信心时（≥3个独立状态标记跨段落出现）才进入报告

## ⚠️ D30-D40 ND confound 映射补充

**问题**：现有的ND1-ND6 confound映射表覆盖了D1-D29的人格维度confound方向，但D30-D40与ND标记同样存在交互——某些ND特征会系统性地偏倚D30-D40的观察，若不校准将在Step 3的D30-D40补充归纳路径中产生错误推断。

**D30-D40 × ND confound 映射表**：

| ND标记 | 影响的D30-D40维度 | confound方向 | 校准规则 |
|--------|------------------|-------------|---------|
| ND2↑（系统化加工） | D30情态动词 → 道义情态↑（"必须""应该"）；D38正当化密度↑（系统化理由堆叠）；D40评价锚定→规则标准主导（"这不符合规则"） | 构象低O/高A（大七） | 当ND2≥●中时，D30中的道义情态不纳入O低分判断；D38的高密度+系统化理由的结构化程度独立于尽责性。O评分至少上调0.5分 |
| ND3↑（注意力切换） | D32完结行为 → 弱完结（话题未收束就跳走）；D33提问策略→高频但发散；D31叙事锚点 → 密度不均（超聚焦领域极高，非兴趣领域极低） | 构象低A（行事拖沓） | ND3↑的弱完结反映注意力模式而非尽责性。D32不纳入A低分判断。D33的发散提问仍可作为O的弱信号但标注"受ND3影响" |
| ND3'↑（超聚焦） | D13通过D31间接影响 → 在超聚焦领域中D31极高、叙事锚点密度异常大 | 构象高A/高I（领域特异而非稳定特质） | 超聚焦领域的高D31不能推广为整体人格。I/A评分一致性判断降一级 |
| ND4↑（masking） | D35评价框架 → 过度使用社会规范标准（"正常人都会……"）；D36认知谦逊标记→ 策略性谦逊（"这只是我的看法"） | 构象低T/高社会从众 | ND4↑的D35/D36信号反映社交脚本分析而非真实认知谦逊。T评分以场景映射为准，D30-D40在T维度上的信号权重降半 |
| ND5↑（认知补偿/视觉空间思维） | D39隐喻源域 → 空间/视觉隐喻偏多（"从全局看""这个框架""路径图"）；D37动作-状态动词 → 视觉化动作描述偏多 | 无统一方向confound，但D39/D37的非语言偏好可能被误判 | ND5↑的隐喻偏好是认知风格而非人格特质。D39隐喻源域偏好的O/开放性评分上调0.5分（确保不因非语言思维被误判为低O） |
| ND6↑（时间盲） | D31叙事锚点 → 时间坐标稀疏（"不知道什么时候""感觉才一会儿"）；D32叙事完结 → 突然截断（时间感知不连续导致叙事组织困难） | 构象低A（叙事组织困难被误判为尽责性低） | ND6↑的时间锚点稀疏≠叙事能力差。D31中时间坐标的缺失不纳入A低分判断 |

**融合规则**（与主ND校准表同时使用）：
- D30-D40 ND confound校准在Step 3的D30-D40补充归纳路径中应用
- 调节后的D30-D40信号在跨路径分歧时标注"该D30-D40维度信号可能受ND confound影响"
- 当ND校准后的D30-D40信号方向与场景映射方向相反 → 以场景映射为准，标注"ND confound下D30-D40信号已校准，但仍非独立于场景映射"

## ⚠️ 虚构创作/同人文本校准层（Fiction / Fanfiction Confound）

**问题**：同人小说、原创故事等虚构文本中，作者的语言通过角色和叙事视角间接呈现。如果直接套用标准流程，会将角色的情感强度误解为作者的N分，或将角色的关系模式误解为作者的R/E分。

**核心校准原则**：

| 维度 | 虚构创作中的信号类型 | 校准规则 |
|------|-------------------|---------|
| **O 开放性** | 隐喻创造力、跨域类比、叙事结构设计 → **可直接观测** | 标准流程，置信度不降 |
| **I 才干** | 语言驾驭能力、心理洞察深度 → **可直接观测** | 标准流程，置信度不降 |
| **N 情绪性** | 角色情感强度、情节情绪张力 → **反映作者可调用的情感范围** | 原始观察降一级解读（■→●），标注"虚构体裁" |
| **A 行事风格** | 叙事结构、章节组织、节奏控制 → **可观测但受体裁约束** | 仅当跨作品一致时才作为A信号，单文本不评 |
| **E/R（人际关系）** | 角色关系模式、角色间互动方式 → **不映射作者自身关系模式** | E/R不评，报告标注"虚构创作无法评估人际关系" |
| **T 处世态度** | 题材选择、角色价值观、冲突类型 → **弱信号，可参考** | 仅标注为探索性（△），不纳入评分 |

**特殊处理——同人vs原创**：
- **同人创作**：使用已有角色和世界观，作者的O/I信号通过角色诠释间接体现。角色本身提供的信息（MBTI类型等）不能作为作者信息使用
- **原创创作**：自创角色和世界观，作者的O/I信号更直接。叙事结构反映A的更多信息

**叠加规则**：虚构创作校准层在文体基线校准之后应用，无论文本多长，E/R均不评。

## ⚠️ 网络求助/咨询语境校准层（Help-Seeking / Consultation Context Confound）

**问题**：向陌生人（博主/咨询师/专家）求助时，语言的礼貌策略和印象管理强度远高于日常社交。具体偏倚方向：道歉频率↑、自我贬抑↑、使用敬语↑、自主表达↓。若不做校准，容易被误读为低E（退缩）、低A（犹豫不决）、低I（表达能力弱）。

**语境标记**（满足≥2条激活校准）：
- 开场白含"你好""打扰了""请问"
- 含"抱歉/不好意思/打扰"等道歉标记
- 问题结尾带"[拔草R]"等表情符号作软化
- 自述含"我觉得""可能""也许"等高密度不确定性措辞
- 在发出请求后主动提供退路（"不方便也没关系""不做朋友也可以"）

**校准规则**：

| 受偏倚维度 | 偏倚方向 | 校准方式 |
|-----------|---------|---------|
| E 外向善良 | 偏↘（主动联系不被视为日常外向的信号） | E评分以求助行为本身为唯一依据→不评，标注"求助语境" |
| A 行事风格 | 偏↘（礼貌策略表现出的犹豫不反映组织能力） | A的偏低分数上调1分 |
| I 才干 | 偏↘（求助者可能刻意弱化能力） | I不降分，求助中的提问质量才是I的真实信号 |
| N 情绪性 | 偏↗（求助本身意味着应激状态） | N的偏高分数视求助内容定——求助内容平和则不调；内容含强烈情绪描述则降1分 |
| R 人际关系 | 偏↘（求助中的退缩策略不反映基线信任水平） | R不评，标注"求助语境" |
| O 开放性 | 无系统偏倚 | 标准流程 |

**叠加规则**：与工作语境、临床状态等其他confound相乘。

## ⚠️ 中英混杂/代码混合文本校准层（Code-Mixing Confound）

**问题**：中英混杂（code-switching / code-mixing）在海外华人、留学生、特定圈层（如同人圈/游戏圈/科技圈）中常见。混合文本的句长、语气词密度、正式度、词汇精确度测量会因语言切换产生系统性偏倚。此外，code-mixing行为本身在某些群体中具有社交身份标记功能（圈层归属），而非人格信号。

**混杂类型区分**：

| 类型 | 定义 | 典型场景 | 校准方式 |
|------|------|---------|---------|
| **功能型混合** | 因术语/概念无对应中文翻译而切换 | 科技/学术用语（"reset一下""bias问题"） | 不校准——术语切换非人格信号，D14/D15排除混杂词后评估 |
| **身份型混合** | 用切换语言标记社群归属 | 留学生/同人圈（"这个真的so sweet""我要去meeting了"） | D15正式度↓调解0.5分（混合说话者不是正式度低，而是在用混合标识身份）|
| **认知型混合** | 因双语认知优势，某概念在另一语言中更易提取 | 双语者自然切换，无规律 | 不校准，但D5断裂句排除语言切换导致的停顿（非自我修正） |

**D维度受偏倚及校准**：

| D维度 | 偏倚 | 校准规则 |
|-------|------|---------|
| D4句长 | 单语段落偏短（混合文本倾向短切换） | 仅统计单语段落的句长，不统计语言交界处的句长 |
| D6语气词 | 偏↘（混合文本倾向于减少语气词使用） | 不调——语气词仅在中文字段中统计 |
| D14用词精确度 | 偏↗（专业术语常见于混合文本） | 排除术语类混杂词后重新评估 |
| D15正式度 | 偏↘（身份型混合的正式度偏低） | 身份型↑调0.5分，功能型不调 |
| D5断裂句 | 偏↗（语言切换处的停顿被误读为自我修正） | 语言切换导致的停顿不纳入D5计数 |
| D25填充词 | 偏↗（语言搜索导致填充词增加） | 填充词按原文语言分别统计，跨语言填充不纳入 |

## ⚠️ 专业领域黑话校准层（Professional Jargon Confound）

**问题**：医学/法律/学术/技术等专业领域写作者（或在专业语境中发言的人），其语言受领域规约的程度可能远大于人格影响。具体表现：句子偏长、因果链清晰、正式度高、情绪词汇低。若不校准，容易被误读为高A、低N、高I——但这些特征更可能来自领域规约而非人格。

**需要校准的场景**：
- 专业论文/报告/技术文档
- 医患沟通中的医学语言
- 法律文本/律师意见
- 学术讨论中的术语密集表达
- 政策文件/制度文本

**校准规则**：

| D维度/人格维度 | 偏倚方向 | 校准规则 |
|--------------|---------|---------|
| D4句长 | 偏↗（学术/法律文本天然长句） | 与正式写作基线对比，超出基线的部分归因于领域要求而非A |
| D12因果链 | 偏↗（论文章需要因果推理） | D12的高强度部分归因于体裁（20-30%），人格相关的因果链仅取中等强度信号 |
| D15正式度 | 偏↗（领域术语+正式文风） | D15的基线调到正式写作的上限。正式度校准值不进入A评分 |
| D9情绪词汇 | 偏↘（专业文本要求克制情绪表达） | D9低分不在非专业语境下推论为低N。N维度以场景映射为准，不以D9为准 |
| D14用词精确度 | 偏↗（专业术语本身就是高精度） | 排除术语后评估日常词汇精度 |
| A 行事风格 | 偏↗（论文/报告的结构性来自发表要求非人格） | A评分下调1分，标注"专业语境" |
| O 开放性 | 可能偏↘（学术要求保守/引用前人） | O以跨领域引用和假设性思维为锚点，不以D4/D12为准 |
| N 情绪性 | 偏↘（专业文本不表达情绪） | N不评（场景稀疏），标注"专业文本" |
| I 才干 | 偏↗（专业写作不等于日常才干） | I仅评除专业知识以外的认知特征（如元认知、类比创造力） |

**叠加规则**：与工作语境校准层可叠加（专业人士在工作场合写作→专业黑话×工作语境→取最低置信度）。

## ■ D4 句子长度与从句复杂度 → 认知复杂度/开放性

## ■ D4 句子长度与从句复杂度 → 认知复杂度/开放性
**置信度: 高**

### 研究基础
- Mairesse et al. (2007), *Using Linguistic Cues for the Automatic Recognition of Personality in Conversation and Text*: 开放性(Openness)与句子长度、词汇复杂度正相关
- Pennebaker LIWC体系: 长词/长句使用者倾向更高的认知复杂度
- 开放性高的个体使用更多感知类词汇、更长词句、更复杂句式

### 核心映射
| 特征 | 对应心理 |
|------|---------|
| 长句 + 多层从句嵌套 | 认知复杂度高、抽象思维强、经验开放性高 |
| 短句 + 简单主谓宾 | 具体思维、直接表达、可能外向性较高 |
| 句长变化大 | 认知灵活性高、能根据内容调整表达 |

### 局限性
- 写作/正式场景中句长普遍偏长，需做场景归一化
- 技术性内容天然需要长句，与人格无关
- 跨语言比较需注意（中文流水句 vs 英文复合句）
- 方言语法迁移：粤语/闽语/吴语等方言的句法结构（如疑问句语序、双宾结构）迁移至普通话时，可能改变句子长度和复杂度，需识别并排除

## ■ D8 句式结构与依恋标记 → 依恋风格
**置信度: 高**

### 研究基础
- Dunlop et al. (UC Riverside), *Social Psychology and Personality Science*: 分析1400+自传叙事，发现依恋风格在代词使用和句法模式上有稳定标记。焦虑型与回避型均呈现高I-talk + 低we-talk
- Nir & Sher-Censor (2024), *Acta Psychologica*: 依恋安全性越高，叙事中从句整合度越高（更多因果连词"因为""所以"、更好的信息流管理）
- Calvo (2018), *Computación y Sistemas*: Word Space Model分类依恋风格，安全型识别准确率85.71%

### 核心映射
| 语言标记 | 对应的依恋风格 |
|---------|--------------|
| 平衡"I"/"we" + 连贯叙事 + 因果连接词 | 安全型 |
| 高"I" + 话题跳跃 + 犹豫/修正 + 高情绪词 | 焦虑型（ preoccupied ） |
| 低"we" + 简短回答 + 避免情感表达 + 低情绪词 | 回避型（ dismissing ） |
| 句法碎片化 + 矛盾表述 + 威胁情境下字数骤变 | 混乱型（ disorganized/fearful ） |

### 局限性
- 需要足够长的叙事文本（最好100词以上）
- 不同关系语境下依恋表现可能不同（朋友 vs 伴侣 vs 家人）
- 自我报告的依恋风格与语言标记的相关性为中等效应量

## ■ D9 人称代词模式 → 自我关注/情绪状态
**置信度: 高**

### 研究基础
- Pennebaker (2011), *The Secret Life of Pronouns*: 人称代词是最强的心理语言标记之一。高"I"使用是抑郁/神经质的最强语言预测因子
- 对连环杀手Jack Unterweger和BTK杀手Dennis Rader的信件分析：异常高"I"使用 + 情绪语调平淡
- 跨文化验证：中英文语境下均成立

### 核心映射
| "我"频率 | 解释 |
|---------|------|
| 极低 | 回避自我暴露、正式距离、可能高权力感 |
| 适中 | 健康的自我关注 |
| 高 | 情绪困扰、神经质、自我沉浸、身体疼痛 |
| 极高 | 抑郁倾向、自恋特质、危机状态 |

| "我们"频率 | 解释 |
|---------|------|
| 高 | 社会连接感、集体主义、安全依恋 |
| 低 | 个体主义、回避型依恋、社会疏离 |

### 局限性
- "我"在中文口语中常被省略（主语脱落语言），需考虑语境
- 群组身份 vs 伴侣关系中的"我们"含义不同，需区分

## ● D1 标点符号规范性 → 严谨性/细节关注
**置信度: 中**

### 研究基础
- Grammarly (2015) "Personality Profiles of 12 Punctuation Marks": 标点习惯与人格特质的关联分析
- *The Spectator* (2015): 标点习惯暴露对"决断力"和"最终性"的心理舒适度
- 网络心理学研究：不用标点 → 效率偏好；极规范 → 完美主义倾向

### 核心映射
| 特征 | 解释 |
|------|------|
| 标点极不规范/无标点 | 效率优先、不拘小节、可能低尽责性 |
| 规范但过度使用（如多次感叹号） | 情绪强烈、完美主义、可能神经质 |
| 标点极度精确 | 高标准、规则遵从、可能强迫倾向 |
| 波浪号~ / 非常规标点 | 亲和需求、非正式、创意表达 |

### 局限性
- 手机输入对标点规范性有系统影响（纯效率因素）
- 不同平台/场景的社交规范不同（微信 vs 邮件 vs 知乎）
- 方言书写习惯：部分方言母语者在书写普通话时标点使用可能迁移口语习惯（如粤语书面语对标点的使用习惯与普通话书面语不同），需注意排除

## ● D2 句号使用模式 → 边界感/情绪闭合
**置信度: 中**

### 研究基础
- *The Spectator* 文章将句号使用与"决断力"和"最终性"的心理舒适度关联
- 青少年社交媒体的语用研究发现：句号在短消息中会被感知为愤怒或被动的攻击性
- 一逗到底 → 避免承诺、迎合倾向、不想显得决断

### 核心映射
| 特征 | 解释 |
|------|------|
| 句号分明、每句闭合 | 边界清晰、果断、Type A倾向 |
| 大量逗号一逗到底 | 边界模糊、避免决断、高亲和需求、怕冲突 |
| 频繁省略号结尾... | 回避强硬结尾、玻璃心、犹豫不决 |
| 混合使用（句号+省略号交替） | 时而决断时而犹豫，可能情景敏感 |

### 局限性
- 聊天场景中句号使用率下降是普遍现象，不完全是个人特质
- 年龄效应：年轻人更倾向不用句号
- 方言/社群习惯：部分网络社群或方言群体有特定的断句习惯（如不使用句号而用空格/换行替代），需排除群体规范的影响

## ● D5 半句/断裂句频率 → 焦虑水平/思维跳跃
**置信度: 中**

### 研究基础
- Entwistle (2025), *Linguistic markers of personality dysfunction* (npj Mental Health Research): 自我打断、不完整句与反刍思维正相关。人格功能障碍的语言痕迹包括紧迫感("我需要""我必须")
- 防御机制的语言映射：高防御者语音流畅度更低，口误更多
- 高频断裂句+高"I"是情绪困扰的强组合信号

### 核心映射
| 特征 | 解释 |
|------|------|
| 句子结构完整、极少断裂 | 思维连贯、情绪稳定 |
| 偶尔断裂后自我修正 | 正常认知监控 |
| 频繁断裂、说话说到一半换话题 | 思维跳跃、焦虑倾向、注意力分散 |
| 断裂 + 修补 + 重说 | 高社交焦虑、完美主义表达 |
| 突然中断不补完 | 回避特定话题，可能触发防御 |

### 局限性
- 口语转录中"断裂句"的识别受转录质量影响
- 某些思维风格（如发散型）天然多断裂，不一定病理

## ● D10 情绪词汇密度与效价 → 情绪调节模式
**置信度: 中**

### 研究基础
- Pennebaker LIWC研究: 情绪词使用是人格的最强预测因子之一
- Entwistle (2025): 负面情绪词高频 → 神经质高；愤怒词多+亲密词少 → 人格功能障碍
- 正向情绪词高频 → 外向性高、宜人性高

### 核心映射
| 情绪词特征 | 解释 |
|-----------|------|
| 极少情绪词 | 情绪表达克制、回避型倾向、可能述情障碍 |
| 负面情绪词为主 | 神经质、负性情绪偏好、抑郁风险 |
| 正面情绪词为主 | 外向性高、积极心态、可能宜人性高 |
| 情绪词多但效价混合 | 情绪觉察高、开放表达 |
| 情绪词密集 + 极端表达 | 情绪调节能力相对有限 |

### 局限性
- LIWC效应量在跨样本研究中不稳定（Martínez-Huertas 2022交叉验证失败）
- 社交媒体上情绪表达有自我呈现偏差

## ● D11 绝对化词汇频率 → 认知灵活性
**置信度: 中**

### 研究基础
- Entwistle (2025): "总是""从不""必须"等绝对化词高频使用是认知僵化的语言标记，与人格功能障碍相关
- Beck的认知三联理论：绝对化要求是抑郁/焦虑的认知特征之一
- 二分法思维（dichotomous thinking）的语言标记

### 核心映射
| 绝对化词密度 | 解释 |
|------------|------|
| 几乎无 | 认知灵活性高、相对化思维、辩证 |
| 偶尔使用 | 正常范围 |
| 频繁（总是/从不/必须/肯定/绝对） | 认知僵化倾向、二分法思维 |
| 极端表达 + 高否定词 | 认知灵活性显著受限 |

### 局限性
- 某些话题天然引发绝对化表达（如道德判断）
- 情绪激动时绝对化词增加，需要多情境样本

## ● D12 否定词频率 → 防御性/对立性
**置信度: 中**

### 研究基础
- LIWC研究: 否定词（不、没、不能、不要）是消极情绪和防御性的标记
- Entwistle (2025): 人格障碍群体否定词使用频率显著高于对照组
- 高否定词 + 高第一人称 → 心理困扰的强组合信号

### 核心映射
| 否定词密度 | 解释 |
|-----------|------|
| 极低 | 接纳性强、肯定倾向、低防御 |
| 适中 | 正常范围、能表达不同意见 |
| 高 | 防御性高、对立态度、拒斥倾向 |
| 极高 + 密集出现 | 对立违抗倾向、可能抑郁/焦虑 |

### 局限性
- 需要排除对话情境中的合理否定（如否认某个事实）
- 辩论/批判性讨论中否定词天然增多

## ● D13 因果/逻辑连接词 → 分析性思维
**置信度: 中**

### 研究基础
- Pennebaker的语言风格分类框架: 因果词（因为/所以/因此）和分析性思维正相关
- 分析性思维风格与高认知需求、阅读量正相关
- 缺乏连接词 → 叙述型思维、高宜人性

### 核心映射
| 因果词密度 | 解释 |
|-----------|------|
| 低 | 直觉型思维、叙述型表达、高宜人性 |
| 中 | 平衡型思维 |
| 高 | 分析型思维、高认知需求、逻辑驱动 |
| 极高 + 其他逻辑词密集 | 可能高智商、理性主导、情绪表达克制 |

### 局限性
- 学术/技术文本中因果词天然高
- 口语对话中因果词使用率低于正式写作

## ▲ D3 段落长度分布 → 思维组织方式
**置信度: 低**

### 基础
基于观察性分析：段落结构能反映思维组织偏好，但缺乏系统性实证研究。

### 核心映射
| 特征 | 可能的解释 |
|------|-----------|
| 大段密集文字 | 全局型思维、连续性强、深度聚焦 |
| 短段 + 频繁换行 | 快节奏思维、碎片化处理、强调点多 |
| 单句成段 | 修辞意识强、可能高焦虑/强调需求 |
| 段落长度均匀 | 组织性强、有结构感 |
| 段落长度忽长忽短 | 思维随内容变化，灵活但可能缺乏结构 |

### 局限性
- 场景影响极大（微信聊天天然短段，邮件天然长段）
- 缺乏直接研究支持，仅为经验观察

## ▲ D6 语气词使用 → 情感状态/亲和需求
**置信度: 低**

### 基础
基于语用学和网络文化观察。中文语气词（啊、吗、呢、吧、哈、嗯、嘛、哟）具有情绪传递和社交调节功能，但缺乏系统量化的人格映射研究。

### 核心映射
| 语气词 | 可能的语用功能 |
|-------|---------------|
| 哈/哈哈/呵呵 | 缓和气氛、亲和需求、尴尬管理 |
| 吧 | 不确定缓冲、寻求认同、软化建议 |
| 嘛 | 理所当然、撒娇、寻求共鸣 |
| 呢 | 反问/好奇、互动邀请 |
| 嗯/哦 | 被动回应、兴趣低、回避深入 |
| 啊/呀/哟 | 情绪外显、惊喜/惊讶、生动感 |
| 语气词总体密度高 | 亲和需求高、情绪外显、不确定感 |
| 完全无语气词 | 正式克制、情感表达保守、高边界 |

### 局限性
- 年龄/性别/地域差异巨大——四川话"撒"、湖南话"咯"、台湾"啦/喔/捏"、东北"整/呗"等方言语气词会被误判为高亲和需求(D6)，需先做方言识别
- 纯研究性不足，基于语用学推断而非人格心理学实证
- 网络语言中语气词的变体（"惹""嗷""捏"等）快速演变

## △ D7 格式塔语言处理倾向 → 高敏感↔低敏感光谱
**置信度: 探索性**

> ⚠️ **实证警告**：本维度的实证基础在近期研究中受到严重质疑。GLP（格式塔语言处理）在语言发展领域的定义和测量方法尚未达成共识。D7 在 LPP 中的定位为**纯探索性观察维度**，不建议在正式分析中作为人格归纳依据。如使用，必须在报告中标注"△ 探索性"并附以下段落中的局限性说明。

### 核心概念
本维度是一个双极光谱，低端对应**低敏感/粗线条型**（五大三粗、不拘小节、对隐性社交信号不敏感），高端对应**高敏感型**（HSP、格式塔加工、深度处理）。

这不是独立的两类人，而是一个连续谱。大多数人落在中间区域。

### 双极对照

| 特征 | 低敏感端（1-4分） | 高敏感端（7-10分） |
|------|-----------------|-----------------|
| 认知风格 | 线性、分析、直奔主题 | 格式塔、整体直觉 |
| 信息处理 | 细节优先→逐步→结论 | 整体印象→再展开细节 |
| **叙述结构** | **分-总结构：细节在前，结论在后** | **总-分结构：结论在前，细节在后** |
| 社交感知 | 对氛围迟钝、直接表达 | 对情绪暗流敏感、谨慎表达 |
| 模糊耐受 | 低——"说重点""总之" | 高——能容纳不确定和歧义 |
| 隐喻使用 | 极少，就事论事 | 丰富，用类比理解世界 |
| 理解方式 | 依赖个人经历和感官体验代入——"我经历过类似的事所以我能懂" | 依赖共情直觉和抽象感知——"我能感觉到你的状态" |

### 研究基础
- 高度敏感者(HSP)由Aron & Aron(1997)提出，约15-20%人口属于高敏感群体，特征包括深层信息处理、易过度刺激、高情绪反应性和对细微刺激的敏感性
- 格式塔语言处理(GLP)源于格式塔心理学——整体优先的认知风格，强调语境依赖和关系感知
- 低敏感端在人格研究中表现为低开放性/低感觉寻求的反面——偏好具体、直接、可操作的信息

### 神经类型与加工风格的探索性关联

注意：以下关联**存在争议且缺乏充分实证支持**，所有输出必须标注探索性。

**ASD领域的核心悖论**：
- 经典认知理论认为ASD偏向**局部加工/弱中央统合**（Weak Central Coherence）——在细节识别任务中表现优异，但整合整体信息有困难
- 语言发展领域观察到部分ASD个体呈现**格式塔语言处理**（Gestalt Language Processing, GLP）——以完整短语/语块（而非单个词）作为语言习得的起点，表现为延迟回声语言、脚本化表达
- Schwartz & Caldwell-Harris (2025, *Frontiers in Psychiatry*)提出两者可调和：都源于**增强感知功能**（Enhanced Perceptual Functioning），听觉通道（语言）和视觉通道（空间）的加工风格可能不一致

**可能与低敏感端/分析端关联的ASD语言特征**（探索性）：
- "细节先于概念"思维：从具体细节逐步建构整体理解，而非先有整体框架
- 字面化表达倾向：隐喻、习语、反讽等非字面语言使用较少或使用方式独特
- 对模糊表述不耐受，倾向明确、具体的措辞
- 某一领域内深度聚焦，话题切换谨慎
- 精确重复特定短语/引用（脚本化语言）

**可能与高敏感端/格式塔端关联的语言特征**：
- 丰富使用类比和隐喻来传达概念
- 对隐性含义和情绪调性敏感
- 自然发散性话题跳跃
- 在模糊信息中做出直觉判断

**重要限制**：
- Haydock et al. (2024, *Autism*)主张接纳格式塔语言发展为神经多样性肯定实践，但Hutchins et al. (2024, *Autism & Dev Lang Impairments*)对GLP概念框架提出强烈质疑，认为其定义模糊、缺乏实证支持
- 单个样本中观察到的"分析型"语言特征可能反映认知风格、专业训练、话题特性或神经类型的综合作用，无法区分
- 加工风格受话题领域、情绪状态、熟悉度等多因素影响
- 听觉（语言）和视觉（空间）通道的加工风格可能不一致

### 探索性假设
1. **高敏感端**：格式塔叙述风格 + 隐喻密度高 + 语境敏感性高 → 可能指向HSP
2. **低敏感端**：线性叙述 + 直奔主题 + 低隐喻 → 可能指向低敏感性人格（粗线条）
3. **中间带**：能灵活切换两种模式 → 认知弹性好

### 重要区分：D7与D13（分析性思维）的关系

D7和D13测量的是**两个独立维度**，评分时必须分开判断：

| 维度 | 测量什么 | 高分的表现 | 与另一维度的关系 |
|------|---------|-----------|----------------|
| D7 敏感度光谱 | 加工风格：整体直觉 vs 线性具体；对隐性信息的敏感度 | 高敏感端：格式塔直觉、隐喻丰富、氛围敏感 | 高敏感的人可以分析性强（"深层分析型HSP"）也可以弱 |
| D13 分析性思维 | 因果/逻辑连接词的密度和因果链长度 | 多因果词、多逻辑连接、解释型思维 | 分析性强的人可以高敏感也可以低敏感 |

**常见误判**：一个习惯用因果分析解释问题的人（D13高分），如果同时呈现分-总结构和低隐喻，可能被错误标记为低敏感端——但实际上ta可能是高敏感+HSP中的"深层分析型"子类型。区分方法：看**叙述结构**（分-总？总-分？）和**隐性信息敏感度**（是否关注氛围/情绪暗流？），而非看是否用了因果词。

## ● D14 用词正式度与敬语频率 → 权力距离感知/社交角色意识
**置信度: 中**

### 研究基础
- **Brown & Levinson (1987)**, *Politeness: Some Universals in Language Usage*: 礼貌三大变量——社交距离、权力差异、强加程度——共同决定语言正式度和敬语选择。这是语用学经典框架。
- **Danescu-Niculescu-Mizil et al. (2013)**, *A Computational Approach to Politeness* (Stanford): 大规模Wikipedia/Stack Exchange数据验证——高权力者更不礼貌，晋升后礼貌度下降；敬语选择与社交地位动态关联
- **簡晉龍等 (台湾本土心理学研究)**, *儒教「尊尊」的體現——權威敏感之運作與社會適應*: 华人社会特有的"权威敏感"人格构念——高权威敏感者更留意权威者在场并恭敬对待
- **Mirzaei et al.**, *Being Polite in Conversation: Power, Distance, and Self-Esteem in Persian Requests*: 低自尊者与高自尊者在礼貌策略选择上显著不同
- **Moon, Uskul & Weick**, *Cultural differences in politeness as a function of status relations* (Journal of Theoretical Social Psychology): 高权力距离文化(韩国)地位显著影响敬语使用，低权力距离文化(英国)则无差异
- **Morand (1996)**, *Dominance, Deference, and Egalitarianism in Organizational Interaction* (Organization Science): 权力嵌入在语言行为中——语言习惯在言说和维持社会等级

### 核心映射

| 语言特征 | 可能的心理解释 |
|---------|--------------|
| 敬语(您/请)与场景高度匹配 | 社交智商高、角色意识灵活 |
| 对所有人极度正式/高频敬语 | 高权威敏感、高权力距离内化、可能社交焦虑 |
| 对所有人极度随意/无敬语 | 平等主义、反权威倾向、低权力距离感知 |
| 敬语使用在不同对象间差异显著 | 社会认知敏锐、等级意识强、适应能力高 |
| 过度礼貌（hyper-politeness） | 低自尊、高社交焦虑、不安全感 |
| 正式词/书面语密集（而非口语词汇） | 高教育水平、正式化人格、情感距离偏好 |
| 立场词混杂（突然从官方式切换到口语化） | 角色同一性探索、可能社交不安 |
| 避免使用任何敬语/头衔 | 反等级、平等主义、可能低宜人性 |

### 中文敬语标记群
- **敬称**: 您、贵姓、贵公司、阁下、前辈、X总/X老师/X工
- **敬辞**: 请、麻烦、劳驾、拜托、承蒙、感谢
- **自谦**: 鄙见、拙见、不成熟的想法、个人认为、斗胆
- **正式词汇选择**: 与口语同表达的正式版本（如"了解"vs"知道了"、"知悉"vs"知道了"、"收到"vs"OK"）
- **句式礼貌**: "如果方便的话"、"不知可否"、"可否麻烦您"

### 局限性
- 中文网络语境中"您"的使用正在减少，年轻人倾向"你"
- 某些行业/组织文化强制要求敬语（如客服、服务业），不反映个人特征
- 敬语使用的权力距离效应受文化强烈调节
- **方言/区域差异**：北方方言区（北京、天津、山东等）日常更常用"您"，南方方言区（粤语、吴语、闽语等）日常对话中"您"的使用频率天然较低——这不等于低权力距离感知，而是语言习惯差异
- 单次互动中的敬语水平不一定反映稳定人格，需跨场景观察

## ● D15 话题跳跃/思维连贯性 → 注意力控制/解离倾向
**置信度: 中**

### 研究基础
- **Andreasen (1986), Thought Disorder Index**: "Tangentiality"（离题）和"derailment"（脱轨）是思维障碍的核心语言标记，表现为话题间缺乏逻辑连接
- **Bernstein & Putnam (1986), Dissociative Experiences Scale (DES)**: 解离体验量表中包含"在谈话或阅读时突然忘记刚才发生了什么"——注意力连续性的断裂是解离的核心特征之一
- **临床区分框架**: 焦虑型重复（perseveration / 同一内容的循环）vs 解离型跳跃（derailment / 线索的中断和切换）在临床语言学中有明确区分——前者聚焦同一主题反复说，后者跳转到无关主题
- **工作记忆与认知控制**: 话题维护需要工作记忆和认知控制，频繁的无关跳跃可能反映执行功能减弱

### 核心映射
| 特征 | 可能的解释 |
|------|-----------|
| 高度聚焦，即使重复也在同一主题上 | 注意力控制好，或焦虑型反刍 |
| 话题正常过渡，有逻辑连接 | 健康水平 |
| 频繁跳跃到无关话题，无过渡 | 分心倾向、解离特征 |
| 跳跃后无法回到原话题 | 解离倾向更强 |
| 被细节分支带跑无法收束 | 注意力控制弱 |

### 与D5的区分
D5（断裂句）关注**句子层面**的中断（不完整句、自我打断），D15关注**话题层面**的跳跃（从一个主题切换到另一个不相关的主题）。一个人可能在句子层面完整（D5低分），但在话题层面跳跃（D15高分）——反之亦然。两者同时高分是更强烈的信号组合。

### 局限性
- 需要足够长的样本来判断话题模式（至少200字以上）
- 某些文化/个人叙事风格天然多话题跳跃（如发散型思维风格），不一定异常
- 咖啡因、疲劳等状态因素会影响话题跳跃，不一定是稳定特质
- 不同话题的跳跃模式差异本身有信息量（什么话题触发跳跃）

## ● D16 用词精确度/具体性 → 思维精确性/认知风格
**置信度: 中**

### 研究基础
- **Semin & Fiedler (1988), Linguistic Category Model (LCM)**: 语言从具体到抽象的层级模型——描述性动作动词（最具体）→ 解释性动作动词 → 状态动词 → 形容词（最抽象）。抽象程度越高，越反映说话者的认知图式而非客观事实
- **Pennebaker LIWC体系**: "洞察词"（realize/understand）、"确定词"（always/never）和"暂定词"（maybe/perhaps）的使用比例与认知加工深度相关
- **词汇具体性效应 (Concreteness Effect)**: 具体词（可感知、可操作）比抽象词加工更快、记忆更牢——高频使用抽象词可能反映抽象认知偏好或认知负荷
- **临床观察**: 抑郁症患者的语言表现出更多"过度概括化"（overgeneral）——使用更模糊、更抽象的语言来描述具体事件（Watkins, 2011, *Depression and overgeneral memory*）

### 核心映射
| 特征 | 可能的解释 |
|------|-----------|
| 用词具体、量化精确、专有名词准确 | 思维精确、注重细节、专业思维 |
| 万能词高频（东西/事情/搞/弄） | 抽象思维偏好，或词汇广度有限 |
| 模糊收尾高频（之类/等等/什么的） | 边界模糊、回避精确承诺 |
| "大概/差不多/好像"高频 | 不确定感、谨慎型认知风格 |
| 同一语义重复时变换精确措辞 | 词汇丰富度、认知灵活性高 |

### 与D14的区分
D14关注的是**正式程度**（敬语/书面语 vs 口语），D16关注的是**精确程度**（具体 vs 模糊）。一个人可以用非常正式的方式说模糊的话（"关于这个问题，我们需要做一些适当的安排"——D14高分，D16低分），也可以用口语化的方式说精确的话（"昨天下午三点我在东门等了你二十分钟"——D14低分，D16高分）。

### 局限性
- 口语对话中模糊词天然高于书面语（场景归一化）
- 某些领域（商务、外交）的策略性模糊不反映认知特征
- 方言中"搞""弄"等万能词的使用频率受方言习惯影响（如粤语"搞"、西南官话"整"）
- 跨语言比较需注意：某些语言的表达习惯天然更精确或更模糊

## 综合交叉验证原则

### 聚合增强
当多个维度指向同一方向时，可信度提升：
- 高"I" + 高否定词 + 高绝对化词 → 强烈的心理困扰信号
- 低语气词 + 高标点规范 + 句号分明 → 高边界/低亲和
- 长句 + 多因果词 + 段落均匀 → 分析性认知风格

### 矛盾处理
当维度之间矛盾时：
- 优先信任高置信度维度
- 矛盾本身可能是意义信息（如高认知复杂度但极短句 = 可能刻意简化）
- 标注为"值得关注的张力"

### 跨样本一致性
- 多份样本中稳定的语言模式比单次样本更有说服力
- 首次接触 vs 熟悉后的语言变化本身也是分析维度

### 重复模式优先原则

**一次是偶然，重复才是特质。**

评分时必须区分两种类型的语言特征：

| 类型 | 定义 | 处理方式 |
|------|------|---------|
| **语境反应** (situational) | 只在某个特定段落或语境中出现一次的特征，如全文唯一一个"总是" | 不纳入评分，标注为偶然 |
| **重复模式** (recurring) | 在文本不同部分反复出现的特征，如每一段都有"我"开头 | 作为稳定信号纳入评分 |

判断方法：在评分前对每个维度的观察标记做**频率分布检查**——
- 该特征集中在某一段落/语境中？→ 可能是话题触发的情境反应，权重降低
- 该特征均匀分布在文本各处？→ 可能是稳定的表达习惯，权重升高

特别警惕**"话题敏感区"的单次信号**：一个人在谈论特定话题（如创伤、冲突）时出现的语言特征不同于日常表达。如果特征只在话题敏感区出现，标注为"话题触发"而非"稳定特质"。

### 反事实评判法（Counterfactual Judging）

评分时的核心心智模型：**不同性格的人说同一件事，方式不同。**

拿到一个样本后，先理解"他在说什么"（内容），再分析"他是怎么说的"（风格）。具体操作为：
1. **设想替换**：想象一个性格相反的人来表达同样的内容——焦虑型 vs 安全型、分析型 vs 直觉型、高边界 vs 低边界——他们会用什么样的措辞、句式、语气词？
2. **对照差异**：实际文本与你设想版本的差异处，就是心理信号所在
3. **排除内容干扰**：一段关于"被朋友疏远"的叙述必然有负面情绪词——这不是信号，是内容。信号在于：同等难过的人中，为什么有人用"我很痛苦"（沉浸式），有人用"这确实不太好受"（分析距离），有人什么都不说（回避）？

反事实评判防止了"内容→判断"的捷径偏差，强迫注意力回到**风格**本身。

## Step 3 方法论：从跨场景模式到心理画像

LPP的Step 3是**归纳阶段**——将Step 2的跨场景定位模式转化为心理画像。归纳遵循以下原则：

### 核心原则

1. **不自上而下套框架**：大五不是"维度聚合"的结果，而是从场景行为模式中观察到的"这人在多个场景中表现一致"的归纳
2. **归纳 ≠ 算术平均**：不计算场景位置的平均值，而是寻找跨场景的一致方向
3. **大五 × 依恋交叉验证**：两者有理论重叠（依恋焦虑≈高N，依恋回避≈低A），独立归纳后对照可以检验信度

> ⚠️ **重要限制**：
> - 大五人格的诊断通常基于自我报告问卷（NEO-PI-R、BFI），LPP提供的是"从语言行为观察归纳的推论"，不等同于标准测评结果
> - 场景光谱归纳的信度取决于命中的场景数量——覆盖场景越多（≥8个），归纳越可靠
> - 涉及角色扮演、表演性内容（如语C病恋）的场景应从归纳数据中排除

---

### 归纳方法：从场景光谱到大五

每个大五维度通过一组**相关场景光谱**来归纳。方法不是对场景位置做算术平均，而是寻找跨场景的**一致性信号**：

| 归纳强度 | 判断标准 |
|---------|---------|
| ■ 强信号 | ≥3个相关场景落在光谱同一端（1-3或8-10） |
| ● 中信号 | 2个相关场景落在同一端，其余居中 |
| ▲ 弱信号 | 仅1个场景落在一端，或各场景位置分散 |
| △ 无信号 | 无相关场景命中，或命中场景皆为表演性语境 |

---

#### O — 开放性（Openness to Experience）

**核心特征**：想象力、审美敏感、智力好奇、情感深度、行为灵活性、价值观多元。
低分端为"固守型"——偏好熟悉、传统、具体、可预测。

**相关场景光谱**：

| 场景 | 光谱 | 1-3端（低O信号） | 8-10端（高O信号） |
|------|------|-----------------|-----------------|
| 表达不确定 | 探索性vs焦虑性 | 焦虑性不确定（"万一…怎么办"） | 探索性不确定（"我在想…会不会…"） |
| 表达决策 | 分析型vs直觉型 | 随机/放弃（"随便"） | 分析比较（"综合来看…"） |
| 表达欣赏 | 具体叙事vs笼统 | 空洞社交（"不错"） | 具体叙事+归因（"因为…"） |
| 表达回忆 | 感官型vs事件型 | 回避怀旧（"不提了"） | 感官细节（"那天阳光…"） |
| 表达幽默 | 观察型vs互动型 | 表情包替代（纯符号） | 观察幽默+讽刺（"XX就像XX"） |
| 表达困惑 | 内求分析vs外求瘫痪 | 瘫痪（"不知道"） | 内求分析（"我困惑的是…"） |

**归纳规则**：
- 表达不确定场景偏探索性端（1-4）+ 表达决策偏分析比较端（1-4）+ 表达欣赏偏具体叙事端（1-4） → **高O**
- 表达幽默偏观察型端（1-4）+ 表达回忆偏感官型端（1-4）+ 表达困惑偏内求分析端（1-4） → **高O**（补充确认）
- 上述场景半数以上无命中或落在两端中间位置 → 不纳入O归纳

---

#### C — 尽责性（Conscientiousness）

**核心特征**：条理性、责任感、成就追求、自律、审慎、可靠性。
低分端为"灵活型"——随性、不拘小节、 spontaneity。

**相关场景光谱**：

| 场景 | 光谱 | 1-3端（高C信号） | 8-10端（低C信号） |
|------|------|-----------------|-----------------|
| 表达承诺 | 具体锚定vs模糊 | 具体锚定+时间（"周五前发"） | 回避承诺（"再说"） |
| 表达承诺 | 主动跟进vs无下文 | 主动跟进进展（"已做完XX"） | 无下文回避（沉默） |
| 表达决策 | 担当型vs摇摆型 | 担当不后悔（"就这个了"） | 后悔反转（"选错了"） |
| 表达顺从 | 释然vs积怨 | 释然向前（"这样也挺好"） | 积怨翻旧账（"每次都这样"） |
| 表达急切 | 行动催促vs情绪急切 | 行动催促（"多久能好"） | 崩溃式急切（"我要疯了"） |

**归纳规则**：
- 表达承诺偏具体锚定端（1-4）+ 表达决策偏担当端（1-4）+ 表达急切偏行动催促端（1-4） → **高C**
- 表达承诺偏回避端（7-10）+ 表达决策偏摇摆端（7-10）+ 表达顺从偏积怨端（7-10） → **低C**

---

#### E — 外向性（Extraversion）

**核心特征**：热情、乐群、果敢、活跃、正性情绪、社交能量。
低分端（内向性）为"独立型"——偏好独处、深度社交>广度社交。

**相关场景光谱**：

| 场景 | 光谱 | 1-3端（高E信号） | 8-10端（低E信号） |
|------|------|-----------------|-----------------|
| 表达欣赏 | 具体叙事vs笼统 | 具体叙事（热情投入） | 空洞社交（敷衍） |
| 告别收尾 | 情感仪式vs利落 | 情感仪式（"今天太开心了"） | 突然切断（"下了"） |
| 表达爱意 | 直接vs含蓄 | 直接强烈（"我好喜欢你"） | 行为暗示（不直接表达） |
| 表达亲密 | 昵称vs回避 | 昵称专属语言（"宝宝"） | 拒绝亲密（"别这么叫"） |
| 表达幽默 | 互动型vs观察型 | 互动调侃（"您又来啦"） | 纯观察（不参与） |
| 表达抱怨 | 叙事宣泄型vs沉默 | 叙事（"我跟你讲今天…"） | 压抑（沉默） |
| 表达共情 | 帮助vs旁观 | 主动帮助（"我能做什么"） | 仅言语（"加油"） |

**归纳规则**：
- 告别收尾偏情感仪式端（1-4）+ 表达亲密偏昵称端（1-4）+ 表达幽默偏互动型端（1-4） → **高E**
- 表达共情偏帮助端（1-4）+ 表达抱怨偏叙事型端（1-4）+ 表达欣赏偏具体叙事端（1-4） → **高E**（社交能量高）
- 多个关系场景中均偏回避/含蓄端（7-10） → **低E**

---

#### A — 宜人性（Agreeableness）

**核心特征**：信任、坦诚、利他、顺从、谦逊、同理心。
低分端为"挑战型"——质疑、竞争性、直接、低妥协倾向。

**相关场景光谱**：

| 场景 | 光谱 | 1-3端（高A信号） | 8-10端（低A信号） |
|------|------|-----------------|-----------------|
| 表达质疑 | 建设性vs对抗性 | 建设性+替代方案 | 人身质疑（"你懂吗"） |
| 表达拒绝 | 高代价vs低代价 | 承担代价（"下次我请"） | 拒绝+指责（"你怎么不早说"） |
| 表达失望 | 直接归因vs压抑 | 直接+理解（"有点遗憾"） | 反向嘲讽（"你忙你忙"） |
| 表达信任 | 修复vs切断 | 解释修复（"说清楚就行"） | 彻底切断（"到此为止"） |
| 表达愤怒 | 事件vs人格 | 针对行为（"这件事不对"） | 针对人格（"你就是自私"） |
| 表达顺从 | 主动vs不情愿 | 主动（"你说得对"） | 不情愿（"随便你吧"） |
| 表达共情 | 认知vs情感 | 认知+验证（"你的感受合理"） | 共情疲劳（"别想太多"） |
| 表达感激 | 归因vs社交 | 归因+回报（"下次请你"） | 回避感谢（沉默） |

**归纳规则**：
- 表达质疑偏建设性端（1-4）+ 表达拒绝承担代价（1-4）+ 表达失望偏直接端（1-4）+ 表达信任偏修复端（1-4） → **高A**
- 表达愤怒偏针对行为端（1-4）+ 表达顺从偏主动端（1-4）+ 表达共情偏认知端（1-4） → **高A**（补充确认）
- 上述场景多偏对抗端（7-10） → **低A**

---

#### N — 神经质 / 情绪稳定性（Neuroticism）

**核心特征**：焦虑、愤怒敌意、抑郁、自我意识、脆弱、情绪反应性。
低分端（情绪稳定性/低N）为"平静型"——情绪稳定、低反应性、抗压力强。

**相关场景光谱**：

| 场景 | 光谱 | 1-3端（低N信号） | 8-10端（高N信号） |
|------|------|-----------------|-----------------|
| 表达不确定 | 探索性vs焦虑性 | 探索性（"我考虑一下"） | 瘫痪性（"你帮我决定"） |
| 表达后悔 | 补偿vs反刍 | 补偿行动（"我现在去"） | 反刍瘫痪（"反复想"） |
| 表达希望 | 重建vs坠落 | 重建（"下次再试"） | 坠落崩溃（"没希望了"） |
| 表达羞耻 | 自嘲vs自贬 | 自嘲接纳（"好蠢啊哈哈哈"） | 自我厌恶（"我讨厌自己"） |
| 表达急切 | 行动vs情绪 | 行动催促（"大概多久"） | 崩溃急切（"我要疯了"） |
| 表达孤独 | 描述vs绝望 | 描述+归因（"因为…"） | 绝望（"没人在乎我"） |
| 表达恐惧 | 安全vs崩溃 | — | — |

**归纳规则**：
- 表达不确定偏探索性端（1-4）+ 表达后悔偏补偿端（1-4）+ 表达希望偏重建端（1-4）+ 表达羞耻偏自嘲端（1-4） → **低N**（高情绪稳定性）
- 表达急切偏行动型端（1-4）+ 表达孤独偏描述端（1-4） → **低N**（补充确认）
- 上述场景多偏瘫痪/反刍/坠落/自贬端（7-10） → **高N**

### 依恋风格归纳：从关系场景簇

依恋风格不从D8单一维度推断，而是从**关系类场景簇的跨场景一致性**归纳：

#### 关系场景簇

| 场景 | 光谱 | 安全型信号 | 焦虑型信号 | 回避型信号 | 恐惧型信号 |
|------|------|-----------|-----------|-----------|-----------|
| 表达爱意 | 依恋浓度 | 有你就好、适度 | 频繁确认（"你爱我吗"） | 回避表达爱意 | 矛盾表达 |
| 表达亲密 | 距离感 | 昵称但适度 | 融合（"没有你不完整"） | 回避亲密（"别这么叫"） | 靠近后撤退 |
| 表达嫉妒 | 占有边界 | 安全信任（"你去吧"） | 频繁确认（"和谁"） | 冷漠不在意 | 忽远忽近 |
| 表达信任 | 被破坏后 | 解释修复 | 反复给机会但持续不安 | 直接切断（"算了"） | 想信任怕受伤 |
| 表达原谅 | 速度 | 即时翻篇 | 有条件（"保证下次"） | 表面原谅+疏远 | 拖延不决 |
| 表达控制 | 失效后反应 | 接受调整 | 情绪升级（"你为什么不听"） | 完全放手/冷漠 | 切断关系 |
| 表达担忧 | — | — | — | — | — |
| 表达道歉 | 结构 | 归因+补偿 | 过度道歉（"都是我的错"） | 条件道歉（"如果…那抱歉"） | 回避道歉 |
| 表达告别 | 情感表达 | 外显不舍但适度 | 悬停（不断追加新话题） | 突然切断 | 矛盾收尾 |
| 表达倾诉 | — | — | — | — | — |
| 解释沉默 | — | — | — | — | — |

#### 归纳规则

1. **关系场景命中数 ≥ 4个** 才纳入依恋归纳
2. 排除表演性语境（如语C中的病恋场景不能作为依恋判断依据）
3. 面具厚度分析：在亲密关系（伴侣、挚友）中的表现权重 > 社交关系中的表现
4. 跨关系一致性：同一人在不同关系类型中的表现越一致，依恋判断置信度越高

#### 四象限定位

```
              回避轴高
                  │
                  │  恐惧型
                  │  (高焦虑+高回避)
  回避型          │
  (低焦虑+高回避)  │
  ────────────────┼──────────────── 焦虑轴
                  │
  安全型          │  焦虑型
  (低焦虑+低回避)  │  (高焦虑+低回避)
                  │
              回避轴低
```

### 大五 × 依恋 交叉验证

大五与依恋风格之间存在理论重叠，交叉验证可同时提升两者的置信度：

#### 理论重叠映射

| 依恋象限 | 预期大五模式 | 重叠机制 |
|---------|-------------|---------|
| 安全型 | 低N + 高A | 情绪稳定+信任倾向 |
| 焦虑型 | 高N + 中A | 关系焦虑是N的核心子维度+高关系投入 |
| 回避型 | 低N(表面) + 低A | 情感抑制+低信任+低顺从他指 |
| 恐惧型 | 高N + 低A | 高焦虑+低信任+关系不稳定 |

#### 验证逻辑

| 交叉验证结果 | 置信度 | 含义 |
|------------|-------|------|
| 大五 & 依恋 指向一致 | ■ 高 | 两个独立归纳路径汇合，结论可靠 |
| 大五 & 依恋 部分重叠 | ● 中 | 基本一致但有细节分歧，需要更多场景覆盖 |
| 大五 & 依恋 冲突 | ▲ 低 | 需要深入分析——可能是样本不足、表演性内容的干扰，或者是真实的内部矛盾 |

#### 交叉验证示例

| 场景 | 大五归纳 | 依恋归纳 | 验证结果 |
|------|---------|---------|---------|
| 低N + 低D8双轴 | 情绪稳定 | 安全型 | ■ 收敛——信度高 |
| 高N + 低D8焦虑 | 情绪波动 | 安全型(表面) | ● 分歧——可能是"高N但有安全关系"，也可能是"安全叙事掩盖底层焦虑" |
| 低N + 高D8焦虑 | 情绪稳定 | 焦虑型 | ● 分歧——"关系特异性焦虑" vs "全局性神经过敏"，需更多语境判断 |
| 低A + 高D8回避 | 低宜人性 | 回避型 | ■ 收敛——信度高 |

## ● D22 信息密度 → 认知风格/思维效率/社交调节

**置信度：● 中**

### 概念定义

信息密度测量的是：在表达同一意思时，**每单位字数所承载的信息量**。区别于：
- **D4（句长）**：一句长句可以密度高（每从句有新信息）也可以密度低（绕圈说同一件事）
- **D16（精确度）**：精确的人可能话少（高密度）也可能话多（低密度但精确）
- **D6（语气词）**：语气词只是低密度的子标记之一，D22更宏观

### 评分方法：占比法

**不采用1-10单轴评分，而是分别统计三种状态的占比：**

1. **逐句/逐段判断**：将文本切分为语义单位（从句或独立分句），逐一标记为高密度、常态、低密度
2. **统计占比**：计算三类各自的比例
3. **跨场景比较**：同一人在不同场景中的密度分布差异

### 判断标准

| 密度 | 标记特征 | 示例 |
|------|---------|------|
| **高密度** | 每个分句引入新信息；极少填充词；修饰语有实质内容（具体数字/精确形容词）；不重复不绕圈 | "抱歉。我的错。下次不会。"（3信息/8字） |
| **常态** | 正常信息-字数比；偶尔有填充但不影响核心信息提取 | "对不起我来晚了，路上堵车，明天我提前出门" |
| **低密度** | 核心意义被大量字数稀释；填充词密集（就/嗯/那个/反正/怎么说呢）；同一意思重复表述；修饰语以纯强调词为主（真的很/超级的/特别的） | "哎呀我真的不是故意的啦，就是刚才那个……你知道的……就反正……我也不是那个意思啦" |

### 核心标记速查

**高密度标记：**
- 无填充词或极少
- 名词/动词密度高（实词占比高）
- 修饰语携带实质信息（"周三下午三点"而非"很快"）
- 并列结构承载不同命题（非重复）
- 从句嵌套但每层有新信息

**低密度标记：**
- 填充词密集：就/嗯/那个/反正/怎么说呢/其实/就是
- 语气词大量出现：啦/哦/喔/哈/嘛/咯
- 同义重复：同一意思说两遍或以上
- 纯强调词：真的很/实在是太/超级无敌
- 悬垂修饰：话说到一半补充无关细节
- 社交润滑剂：你懂吧/你知道/是不是/对吧

### 心理映射

| 密度模式 | 可能的心理指向 |
|---------|---------------|
| 持续高密度（高密>50%，低密<15%） | 思维精确型、认知效率高、可能内向型/书面型思维 |
| 持续低密度（低密>40%，高密<20%） | 关联型思维、高外向性、口语化认知、可能ADHD倾向 |
| 均衡型（高密30-40%，低密20-30%） | 认知灵活、能根据需求调节密度 |
| 密度随语境剧烈波动 | 高社交认知、情境敏感性强 |
| 高密度+高D4 | 精密复杂型——学者/写作者的典型模式 |
| 低密度+低D4 | 口语社交型——高外向性的典型模式 |
| 低密度+高D4 | 探索型冗长——边说边想、认知探索中 |

### 方言校正

- 口语习惯迁移（如粤语/闽语区书写中语气词迁移）会导致低密度假象
- 手机打字场景（大量断句、少修饰）可能导致高密度假象
- 正式写作 vs 即时聊天的基线差异极大，必须场景归一化

### 在报告中的呈现

```
● D22 信息密度
   高密度 ████████░░  38%  ▏低密度 ████░░░░░░  19%  ▏常态 43%
   ─ 上下文敏感型

   [简短分析：跨场景的密度差异与分布特征]
```

### 局限性

- 口语天然比书面语密度低——跨场景比较时必须考虑媒介差异
- 话题专业性影响密度：专业话题密度高、社交话题密度低，不一定是心理差异
- 聊天软件中短句频繁发送不等于高密度——需区分"媒介导致的短"和"认知风格的高密度"
- 同义反复在某些文化中是礼貌策略（如中文的"哪里哪里"），而非低密度信号

## 补充维度（D17-D21）

### ● D17 归因风格 → 宜人性/神经质
**置信度：中**

**研究基础**：
- **Abramson et al.（1978）, 习得性无助归因理论**：归因的三维度（内部-外部、稳定-暂时、整体-具体）是抑郁的核心认知机制。内部+稳定+整体归因 → 抑郁倾向
- **Seligman（1990）, 解释风格模型**：对好事的内部归因（"我做到了"）+ 对坏事的外部归因（"这次运气不好"）→ 乐观解释风格；反之为悲观解释风格
- **Mezulis et al.（2004）**, 自我服务偏差的元分析：正常人倾向于将成功归因于自己、将失败归因于外部——自我服务偏差减弱是抑郁的标记

**核心映射**：

| 特征 | 可能的心理指向 |
|------|--------------|
| 成功归内因+失败归外因（自我服务正常） | 心理健康、乐观解释风格 |
| 失败归内因（"是我没做好""都是我的错"） | 内归因→抑郁倾向、高宜人性（过度承担）、低N的自我保护不足 |
| 成功归外因（"运气好""别人帮的"） | 低自尊、冒名顶替综合征、低自信 |
| 失败归外因（"他害的""环境不行"） | 外归因→低宜人性、敌对归因偏差、低责任感 |
| 归因模式跨语境不一致 | 情境敏感、认知灵活性；或归因风格尚未固化 |
| 频繁使用"被"字结构（被动语态）归因 | 弱化自身主体性——"被开除了"vs"我辞职了" |

**评分标准（1-10）**：
- 1-3：自我服务偏差正常——成功归内、失败归适度的外/情境因素，解释风格乐观
- 4-6：归因模式混合或不明——无明显偏向
- 7-8：偏向某一端（极端内归因或极端外归因），在多个语境中一致
- 9-10：归因模式僵化+极端化——所有事情归因于同一因素（"都是我的错"或"都怪别人"）

**局限性**：
- 需要同时包含成功和失败事件的文本来做归因对比
- 单次叙述中只有单一归因（如只描述失败）时，无法区分自我服务偏差的完整性
- 文化差异：东亚文化中内归因失败（"我没做好"）在社交层面更受鼓励，不一定是抑郁信号

### ● D18 承诺/履诺语言 → 尽责性
**置信度：中**

**研究基础**：
- **Gollwitzer（1999）, 实施意向（Implementation Intentions）**：具体实施意向（"我将在X时在Y做Z"）比笼统目标（"我要做Z"）的任务完成率高2-3倍
- **语言行为与后续行为的一致性研究**：承诺的表述具体度与实际履行率正相关——"我周五下午三点前发给你" vs "我尽快发给你"
- **计划谬误（Planning Fallacy, Kahneman & Tversky 1979）**：低估任务时间的高频语言标记（"很快""马上""一会儿"）与低尽责性相关

**核心映射**：

| 特征 | 可能的心理指向 |
|------|--------------|
| 具体承诺（明确时间+交付物+责任人） | 高尽责性、计划性强、可信赖 |
| 模糊承诺（"尽快""回头""再说"） | 低尽责性、拖延倾向、承诺回避 |
| 承诺+跟进（"我说到X还没完，现在在Y"） | 高尽责性+高自律 |
| 承诺后无跟进/遗忘标记 | 低尽责性、可能ADHD特征 |
| 补救承诺（"我忘了，我改到明天"） | 中等尽责性——有补救意识但执行力不够 |
| 回避承诺（"我不确定能不能做""看情况"） | 低尽责性、高宜人性（怕让人失望而不敢答应） |

**评分标准（1-10）**：
- 1-3：几乎无承诺表述或全为模糊承诺（"再说""看情况""尽量"）
- 4-6：有承诺但部分模糊——部分有具体时间，部分含糊
- 7-8：大部分承诺具体且有时间锚定，偶有遗漏
- 9-10：承诺高度具体+有跟进/完成标记，极少遗漏

**局限性**：
- 仅适用于包含承诺/任务话语的文本——无相关内容时不纳入评分
- 社交场景中的模糊承诺可能是礼貌策略而非尽责性问题（"改天一起吃饭"=社交客套）
- 需要区分"承诺"的语境（工作约定 vs 社交客套 vs 亲密关系的安抚性承诺）

### ▲ D19 时间锚定密度 → 尽责性
**置信度：低**

**研究基础**：
- **时间人格（Time Personality）研究**：个体在时间取向、时间管理、时间态度上的差异与尽责性高度相关
- **具体时间参考频率**：高尽责者在叙事中使用更多具体时间参考（"周三下午三点" vs "前几天""之前"）
- 缺乏直接的大规模LPP验证——当前为基于观察的推论

**核心映射**：

| 特征 | 可能的心理指向 |
|------|--------------|
| 具体时间锚点多（精确日期/时间/时段） | 高结构化、计划性强、时间管理好 |
| 模糊时间参考为主（"之前""那会儿""有一天"） | 低结构化、时间感知模糊、可能低尽责 |
| 无时间参考 | 叙事脱离时间线——可能是抽象思维或时间组织弱 |
| 时间锚点与实际事件不符（叙事混乱） | 时间记忆混淆——可能疲劳/ADHD/认知负荷高 |

**评分标准（1-10）**：
- 1-3：极少具体时间参考，全为模糊时间词（"以前""后来""有次"）
- 4-6：偶尔有时间锚点，大体可追踪时间线
- 7-8：时间参考较多，大多数事件有时间定位
- 9-10：高度结构化时间线，精确到日期/时段/顺序

**局限性**：
- 回忆性叙事天然比计划性叙事的锚点少——需场景归一化
- 跨语言/文化差异显著——某些文化的时间参考习惯天然更模糊
- 样本量需求高：短文本中时间参考出现频率不足以支撑评分

### ● D20 假设性思维密度 → 开放性
**置信度：中**

**研究基础**：
- **反事实思维（Counterfactual Thinking）与开放性的关联**：高开放性者更频繁且自发地产生反事实思考（"如果当初……会怎样"）
- **条件句使用频率**：在叙事中使用条件句（"如果/假如/假设"）的频率与认知探索意愿正相关，与封闭性负相关
- **创造力研究**：发散性思维任务中表现好的人，在自发言语中产生更多"可能性探索"句式（"或许""可能""另一种可能是"）
- 注意区分"如果"的条件使用（"如果下雨就不去了"=条件规划）vs "如果"的探索使用（"如果我是总统就……"=假设性探索）

**核心映射**：

| 特征 | 可能的心理指向 |
|------|--------------|
| 高频假设性条件句（探索性"如果/假如"） | 高开放性、认知探索活跃、发散思维 |
| "可能/或许/另一种可能是"在非确定语境下使用 | 高开放性、模糊耐受性强 |
| 反事实思考（"如果当时……就好了"） | 高开放性+高N（如果是负面反刍）或高开放性+健康（如果是建设性反事实） |
| 极少假设性语言，只谈已发生事实 | 低开放性、具体思维、偏好确定 |
| 对假设性问题的排斥（"这个假设没意义""别想这些没用的"） | 低开放性、模糊不耐受、实用导向 |

**评分标准（1-10）**：
- 1-3：几乎无假设性语言，叙事限制在已发生事实内
- 4-6：偶尔使用假设/条件句式，多为条件规划（"如果你来我就……"）
- 7-8：频繁使用探索性假设句式，自发产生反事实思考
- 9-10：大量假设性探索，抽象可能性推演，思维实验丰富

**局限性**：
- 话题选择和场景类型极大影响假设性语言频率（技术讨论 vs 日常闲聊）
- 焦虑性"如果"（"如果失败了怎么办"=担忧）与开放性"如果"（"如果试试另一种方法呢"=探索）需区分
- 中文中"如果"的使用率低于英文"if"——需要中文基线校准

### ▲ D21 表达节奏 → 情绪状态/认知风格
**置信度：低**

**研究基础**：
- **言语节奏与情绪 arousal 的关系**：语速（反映为逗号间隔密度）与情绪唤起水平相关——高情绪 arousal 状态下语速加快、停顿减少（Scherer, 1986）。文本中的逗号间隔可以反映类似的"言语呼吸节奏"——短间隔密集表达可能与高情绪投入相关，长间隔缓慢节奏可能与认知分析或情绪抑制相关。
- **句长分布与认知加工风格**：短句的突然插入打断节奏流，起到强调、转折、收束等语用功能。短句的使用频率和位置反映了表达者的修辞意识和情绪控制能力——能通过句长变化来调控叙事节奏是高语言能力的标志。
- **停顿与思维深度（Gould, 1980）**：书面表达中的逗号间隔可作为"认知停顿"的近似标记——在需要更复杂认知加工的内容上，人们倾向于使用更长的句子间隔（即逗号前更多字），因为需要在句子内完成更多信息编码。
- **节奏与人格的初步关联（Pennebaker, 2011）**：个人表达中的节奏模式与人格的某些层面相关——高外向性者倾向于更快的语速/更短的表达间隔，高审慎性者倾向于更规整的节奏。但这主要基于口语研究，书面节奏的对应关系尚待进一步验证。

**核心观察**：

| 特征 | 可能的心理指向 |
|------|--------------|
| 逗号间隔极短且均匀（3-5字/间隔） | 口语化认知风格、可能高情绪arousal、随兴表达 |
| 逗号间隔长且均匀（15-30字/间隔） | 书面化认知风格、计划性强、正式表达 |
| 逗号间隔变化幅度大 | 情境敏感度强、节奏随内容主动调节、修辞意识高 |
| 高频极短句（1-3字）用于转折/强调 | 修辞策略成熟、情绪控制力强、有戏剧感 |
| 不同语境间节奏差异显著 | 高社交调节能力、面具厚度大 |
| 所有语境节奏风格高度一致 | 稳定的认知风格、低社交监控 |

**与D3/D4的区分**：
- D3（段落长度）：关注段落的物理长度（行数/字数），即"一段写多少"——D21关注的是段落长度变化的节奏模式，即"什么时候写长段、什么时候写短段"以及"长短切换的频率"
- D4（句子长度）：关注句子的语法复杂度（从句嵌套、修饰语密度），即"一个句子能装多少信息"——D21关注的是逗号间隔的节拍感，即"一口气说多少字"，二者可以不一致（一句长句可由多个短间隔组成）

**局限性**：
- 逗号使用有较强的写作习惯成分，不一定反映自然节奏——某些人习惯多用逗号，某些人少用
- 不同文体（小说、论文、聊天）对节奏的约束不同，跨文体比较需谨慎
- 节奏是书面文本中研究最少的维度之一，当前标注为探索性

## ▲ D23 社会参照密度 → 社会比较倾向/群体认同
**置信度：低**

### 概念来源

社会参照密度测量的是文本中提及第三方的频率和性质——包括具体人名、关系标签（"老板""朋友"）和泛指他人（"别人""有些人"）。这个维度的理论基础来自：

- **社会比较理论（Festinger, 1954）**：人类有内在驱动力通过与他人比较来评估自己的意见和能力。语言中的社会比较标记频率与比较倾向正相关
- **参照群体理论（Hyman, 1942; Merton, 1957）**：个体在评价自我时选择的参照群体反映了其社会认同结构和价值取向
- **Fenigstein et al.（1975）, 自我意识量表**：公共自我意识（public self-consciousness）高的人在语言中更多提及"别人怎么看我"

### 核心映射

| 特征 | 可能的心理指向 |
|------|--------------|
| 第三方提及频率高 + 语气多为社会比较 | 社会比较倾向高、公共自我意识强 |
| 第三方提及频率高 + 语气温暖/归属 | 社会连接感强、集体主义倾向 |
| 第三方提及频率低 | 低社会比较倾向、独立型自我构念 |
| 第三方语气以指责/贬低为主 | 敌对归因偏差、可能低宜人性 |
| 群体归属标记（"我们XX人"）频繁 | 群体认同高、集体主义 |

### 局限性
- 话题本身会触发第三方提及（如"聊团队协作"天然高频率）
- 社交媒体和论坛文本参照密度天然高
- 当前缺乏直接的语言-人格映射研究

## ▲ D24 不确定性回避标记 → 认知风格/模糊耐受
**置信度：低**

### 概念来源

- **不确定性回避（Uncertainty Avoidance, Hofstede, 1980）**：文化层面的概念，近年被引入个体差异研究——个体在模糊/不确定情境中的不适程度
- **模糊不耐受（Intolerance of Uncertainty, Freeston et al., 1994）**：广泛性焦虑的核心认知特征，表现为对不确定情境的负面认知和情绪反应
- **认知闭合需求（Need for Cognitive Closure, Webster & Kruglanski, 1994）**：对确定答案的渴望和对模糊的回避，与"确定性词汇"密度正相关

### 核心映射

| 特征 | 可能的心理指向 |
|------|--------------|
| 确定性词汇（"肯定""绝对""一定"）密集 | 高认知闭合需求、低模糊耐受、可能焦虑 |
| 可能性词汇（"可能""也许""大概"）密集 | 高模糊耐受、谨慎型认知风格 |
| 确定性:可能性比例均衡 | 灵活型——能根据情境选择措辞 |
| 面对不确定时强行确定 | 低模糊不耐受——需要用确定性缓解焦虑 |
| 面对不确定时探索性应对 | 高模糊耐受——认知弹性好 |

### 局限性
- 专业领域知识会影响确定性措辞（专家天然更多"肯定"）
- 正式写作中可能性词汇更多是一种修辞习惯
- 中文"可能"的使用频次要低于英文"maybe/perhaps"——需中文基线

## ▲ D25 话语标记/填充词功能 → 认知加工/社交调节
**置信度：低**

### 概念来源

- **话语标记理论（Schiffrin, 1987, *Discourse Markers*）**：话语标记在口语中具有组织话语结构、标记说话者态度、管理互动关系的功能。典型标记："well""you know""I mean"（英）/"就""其实""反正""那"（中）
- **填充词与认知负荷（Clark & Fox Tree, 2002）**："um""uh"等填充音是说话者在计划后续话语时的"购买时间"信号——填充词密度越高，认知负荷或表达难度越高
- **Fox Tree & Schrock（2002）**：中文填充词"那个"在口语中承担与英文"uh"类似的认知计划功能
- **功能分类框架**：LPP采用四功能分类——填补空白（认知计划）、犹豫（不确定表达）、缓和（社交调节）、话题转换（话语组织）

### 核心映射

| 填充模式 | 可能的心理指向 |
|---------|--------------|
| 填补空白型为主（"嗯……那个……"） | 较高认知负荷或表达性思维 |
| 缓和型为主（"其实""反正""差不多"） | 高社交调节意识、冲突回避倾向 |
| 自我修正型为主（"不是……我是说……"） | 完美主义表达、社交焦虑 |
| 填充词密度随话题敏感度剧增 | 特定话题触发表达困难或焦虑 |
| 填充词密度在所有话题中一致低 | 表达流畅、思维清晰、低社交焦虑 |
| 填充词密度在所有话题中一致高 | 口语化思维、习惯性不确定表达 |

### 与D22的关系
D25关注填充词的**功能分类**（"用来做什么"），D22关注填充词的**密度**（"有多少"）。高填充词密度但功能单一（如全是填补空白）的意义不同于高填充词密度但功能多样（能灵活用于缓和+组织话题）。

### 局限性
- 中文填充词研究远少于英文，功能分类框架主要基于经验分析
- 填充词使用有极强的习惯成分——有些人天生多用"就"和"其实"
- 口语和书面语中的填充词基线不同，口语中填充词更多但功能更丰富
- **方言干扰显著**：部分方言（如东北话的"整"、四川话的"撒"）在口语中被频繁使用，可能被误判为填充词或语气词

## ● 大七人格（中文输入的主归纳模型）

### 背景

大七人格（王登峰、崔红等，基于中文人格特质词汇系统）是运用词汇学方法对中国人群的人格特质词进行系统因子分析后提出的本土人格模型。与西方大五的关键差异在于：
- 大五来源于英文 trait words 的因子分析
- 大七来源于中文人格特质形容词的因子分析，从初始词库到因素结构均为中文语境独立建构
- 两者有重合但也有差异——大七捕捉到中文语境中独特的心理维度：如"处世态度"（T，淡泊vs功利——在大五中无直接对应维度）、人际关系中的"善良"面（在大五的宜人性中未独立成因素）、以及行事风格中"爽快vs拘谨"这一独特面

### 在LPP中的定位

LPP根据输入语言选择人格归纳模型：

| 输入语言 | 主模型 | 副模型（参考） | 原因 |
|---------|-------|--------------|------|
| **中文**（普通话/粤语等） | **大七人格**（王登峰、崔红等，基于中文人格特质词汇系统） | 大五人格（通过大七→大五映射） | 大七基于中文人格特质词汇的因子分析，维度结构更能捕捉中文语境的心理特征 |
| **欧美语言**（英/法/德/西等） | **大五人格** | 无 | 大五是欧美语言起源的模型，在跨文化样本中有充分验证 |

## ▲ GFP聚合理论（General Factor of Personality）

### 概念

GFP（人格一般因子）是人格心理学中一个具争议性的构念——类比于智力的g因子，GFP是大五各维度之间的共同方差。Musek（2007）首先提出：当大五维度被二阶因子分析时，一个单一因子解释了大部分变异。

### 争议双方

| 立场 | 代表观点 | 证据 |
|------|---------|------|
| **支持：GFP是社会适应性** | GFP高分者情绪稳定+宜人+负责+外向+开放——这是"好人格"的综合指标 | 跨文化稳定存在；与自我监控、情商正相关 |
| **反对：GFP是社会赞许性** | GFP高分者在自陈量表中全线高分，可能反映的是自我呈现偏差而非真实人格 | 他评与自评的GFP相关低于单维度；与社交赞许性量表强相关 |

### 在LPP中的使用

LPP不解决GFP的理论争议，而是将其作为**聚合统计工具**使用：

```
GFP = f(大五各维度的共同方向)
    └─ 如果多数维度指向"适应良好"方向（低N+高A+高C+高E+高O）→ 高GFP
    └─ 如果多数维度指向"适应挑战"方向（高N+低A+低C+低E+低O）→ 低GFP
    └─ 如果方向不一致（如高N+高A+高C）→ 低GFP或无法聚合
```

### 报告要求
- GFP必须在报告中标注为"探索性聚合"，并注明"GFP的理论地位存在争议"
- 不可作为独立诊断依据
- 仅在≥4个大五维度有归纳信号时计算GFP

## ▲ LLM验证层理论

### 概念

LLM验证层是一种**成稿后的反事实交叉验证方法**，利用独立的LLM调用对LPP报告草稿进行检验。其灵感来自：

- **反事实思维（Counterfactual Thinking, Kahneman & Miller, 1986）**：人类认知中"如果……会怎样"的心理模拟是检验判断可靠性的自然方式
- **对抗验证（Adversarial Validation）**：机器学习中用于检测训练/测试分布漂移的方法——如果验证模型无法区分预测和真相，预测可信度高
- **调查者三角化（Investigator Triangulation）**：质性研究中多个研究者独立分析同一数据后交叉核对，以提高信度

### 为什么需要LLM验证层

LPP的自然语言分析由LLM执行，存在以下限制：
1. **自我确认偏差**：LLM在生成分析后，可能在后续检查中倾向于维持原有判断而非质疑
2. **模式识别盲区**：LLM可能过度关注某些模式而忽略其他模式——独立验证可以从不同"角度"观察同一数据
3. **置信度校准问题**：LLM在分析模式（Step 1-3）和报告生成模式（Step 4）中可能使用不同的"标准"，独立验证有助于校准

### 方法论原则

| 原则 | 说明 |
|------|------|
| **独立性** | 验证LLM不应与原分析LLM共享上下文——需要"冷启动"，避免上下文污染 |
| **对称性** | 验证LLM的提示语应同样提供"确认""质疑""补充"的出口——不预设验证会发现问题，也不预设不会发现问题 |
| **证据优先** | 验证LLM的质疑需要基于原文中的语言模式，而非基于内容层面或直觉 |
| **分歧保留** | 如果验证LLM与原分析存在分歧，报告应在第二部分呈现分歧而非隐藏 |

### 在LPP流程中的位置

```
Step 1 → Step 2 → Step 3 → Step 4(草稿) → [LLM验证] → Step 4(终稿)
                                            ↓
                                    不改变Step 1-3结论
                                    仅检验Step 4报告
```

LLM验证层不返回Step 1-3修改分析结论——它只检验Step 4报告草稿是否完整、平衡。如果发现分歧，回到Step 3检查证据链，而非直接在Step 4修正措辞。

---

### 评分流程

1. **先完成第一层评分**：按正常流程对16个LPP维度+D17-D21（如有数据支撑）逐项评分
2. **再聚合到大五**：根据映射关系，将下属维度分数加权汇总：
   - 每个大五维度 = Σ(下属维度分数 × 对应权重) ÷ Σ(权重)
   - 权重：高=3，中=2，低=1
3. **标注聚合置信度**：
   - 高置信度（■）：下属维度中有≥2个■高置信度且其余≥中
   - 中置信度（●）：下属维度以●中为主
   - 低置信度（▲）：下属维度以▲低为主或样本数据不足
4. **在报告中呈现**：大五聚合解读放在维度评分之后，作为"高层画像"独立段落。必须标注"基于语言观察的聚合推论，不等同于标准大五测评"

### 报告撰写措辞

| 聚合置信度 | 措辞示例 |
|-----------|---------|
| ■ 高 | "语言特征反映出较高的开放性——句法复杂、隐喻丰富、假设性思维活跃" |
| ● 中 | "语言特征显示可能偏向内向性——社交引用较少、语气词使用克制" |
| ▲ 低 | "有限的文本提示也许倾向于低尽责性——时间锚点模糊、承诺表述笼统，但样本量不足以确认" |

### 重要限制（必须标注在报告中）

大五人格维度是从语言行为观察中衍生出的聚合推论，而非标准化的心理测评结果。LPP的分析方式决定了：
1. **覆盖范围受限**：某些大五层面（如C的"成就感需求"、E的"寻求刺激"）在普通文本中缺乏可观察的语言标记，无法评分
2. **语境偏差**：不同语境下的大五表现不同，文本分析捕捉到的是"该语境下的表达特征"，不一定是跨语境的特质
3. **不替代测评**：LPP的大五聚合不能取代正式的人格量表（如NEO-PI-R、BFI）
4. **样本量门槛**：大五聚合需要至少300字以上的多语境样本，否则置信度自动降一级

## ▲ D26 施虐/幸灾乐祸词汇 → 暗黑人格（Dark Tetrad-Sadism）
**置信度：探索性**

### 研究基础

- **Paulhus & Williams（2002）, *The Dark Triad of Personality***：自恋（Narcissism）、马基雅维利主义（Machiavellianism）、精神病态（Psychopathy）三者的概念框架。后扩展为Dark Tetrad，加入施虐癖（Sadism）——以他人痛苦为乐
- **Plouffe et al.（2017）, *The Dark Tetrad in Text***：语言分析研究发现，暗黑人格高分者在自然语言中表现出一致的语言标记，包括更多负面情绪词、更多第一人称单数、更少社会参照
- **Sagioglou & Greitemeyer（2016）, *Schadenfreude and personality***：幸灾乐祸（Schadenfreude）与暗黑人格特质显著正相关——在他人不幸中体验到愉悦
- **arXiv 2026 计算语言学研究**：使用LLM对Reddit评论进行暗黑人格标记检测，发现幸灾乐祸词汇（"deserved""serves you right""karma"）与Dark Tetrad自评得分的中等相关（r≈0.3-0.4），但效应量不足以作为独立诊断标记
- **Purdue 2025 中文社交媒体分析**：中文语境中"活该""真惨哈哈哈""笑死"类表达在他人负面事件语境中的频率与自恋和精神病态特质的相关性为小到中等效应量（r≈0.2-0.3），且受情境调节——在竞争语境中这些表达的正常化程度更高

### 核心映射

| 特征 | 可能的心理指向 |
|------|--------------|
| 在他人负面事件语境中出现正面情绪词（"活该""哈哈哈该"） | 幸灾乐祸倾向→可能与低宜人性+高精神病态相关 |
| D26高频（≥3处跨语境）+ D28高频（共情空洞） | 强组合信号→可能指向冷酷特质 |
| D26仅出现在"敌人/竞争对象"的负面事件中 | 情境性幸灾乐祸→可能在正常范围内（竞争性幸灾乐祸） |
| D26出现在所有人（包括中立对象）的负面事件中 | 泛化幸灾乐祸→更值得注意的标记 |
| 幸灾乐祸伴随推理论证（"这就是因果报应"） | 合理化幸灾乐祸→可能指向马基雅维利主义风格 |

### 局限性
- 幸灾乐祸在特定文化/社群中可能被正常化（如竞技体育、网络论战）
- 幽默语境中的"笑死""活该"可能是语言游戏而非真实态度
- 单次文本中出现的孤立幸灾乐祸表达不足以作为人格推断依据
- 中文"笑死"在2023-2026年间已高度泛化——多数使用与真实幸灾乐祸无关

## ▲ D27 工具性人称模式 → 暗黑人格（Dark Tetrad-Machiavellianism）
**置信度：探索性**

### 研究基础

- **Machiavellianism的语言标记（Jones & Paulhus, 2009）**：高马基雅维利主义者在语言中倾向于将他人视为实现目的的工具——表现为工具性人称使用（"你帮我""给我"）而非关系性人称（"我们一起"）
- **Holtzman（2017）, *Facing the Dark Triad***：暗黑人格高分者在自然语言中使用更多强制性动词（你必须/你应该/你给我），更少合作性代词（我们）
- **中文语境中的工具性人称研究（Wang et al., 2024）**："给我+动词"结构在中文命令式中具有鲜明的权力不对称标记——高频使用该结构与自我报告的马基雅维利主义得分呈弱正相关（r≈0.25）

### 核心映射

| 特征 | 可能的心理指向 |
|------|--------------|
| 工具性"你"（"你帮我"）高频且无礼貌策略 | 工具性关系认知→可能与马基雅维利主义相关 |
| 工具性人称（你给我/你必须）在所有语境中一致高频 | 泛化工具性思维模式→更值得注意 |
| 工具性人称随权力关系变化（对下级用工具人称，对上级不用） | 马基雅维利主义的社交策略性——高自我监控 |
| 工具性人称低频且伴随礼貌策略 | 正常范围→不指向工具性关系认知 |
| "我们"作为施加义务的工具（"我们要……""我们得……"） | 隐性工具性人称→责任推卸/义务施加策略 |

### 局限性
- 某些职业/权力位置天然需要命令式表达（如项目经理、团队领导），不反映人格
- 中文"给我"在某些方言中有非命令式用法（如"给我看看"=让我看看）
- 工具性人称的分布在亲密关系和权力关系中基线不同
- 单一样本中观察到的工具性人称可能完全由语境驱动

## ▲ D28 共情空洞标记 → 暗黑人格（Dark Tetrad-Psychopathy/Callousness）
**置信度：探索性**

### 研究基础

- **Hancock et al.（2013）, *Language of Psychopathy***：对精神病态罪犯的语言分析发现——在描述自己的暴力行为时使用分析性距离（"受害者当时"vs"我"）、较少情绪词、较多过去时态（心理距离化）
- **Dunsmore & Halberstadt（1997）, *Emotional Language and Empathy***：共情能力与情绪词使用正相关——低共情者在他人情绪语境中产生更少的情绪词
- **Entwistle（2025）, *Linguistic Markers of Personality Dysfunction***：人格功能障碍群体在他人遭遇负面事件时表现出显著更低的情绪词密度，更多的"客观化"描述
- **arXiv 2026**: 在他人痛苦语境中缺乏情绪回应（共情空洞）是人格功能障碍的最强语言预测因子之一（AUC≈0.72），但该结果基于Reddit英文数据，中文语境下的效应量未知

### 核心映射

| 特征 | 可能的心理指向 |
|------|--------------|
| 他人痛苦语境下情绪词密度降至零 | 共情空洞→可能与情感冷漠/低共情相关 |
| 以分析距离替代共情（"从客观角度……""这个案例显示……"） | 分析性防御→避免情绪卷入 |
| 以解决问题替代共情（直接给建议而非先做情绪确认） | 工具性共情→认知共情存但情感共情弱 |
| 以话题转移回避共情语境 | 回避型共情空洞→可能源于不适而非冷漠 |
| 共情空洞仅对特定对象出现 | 选择性共情空洞——关系特异性，非稳定特质 |
| 共情空洞+高D26（幸灾乐祸）+低D9（情绪词稀疏） | 强组合信号→更值得注意的模式 |

### 与D9的关系
D9测量全局情绪词密度——在某一段落中D9突降至基线的特殊低点，同时该段落出现在他人痛苦语境中，是D28共情空洞的探测方法之一。

### 局限性
- 文化差异：某些文化鼓励"解决问题优先于共情"的回应模式（如工具性共情在东亚文化中更常见）
- 性别社会化：男性被社会化为更少表达情绪共情——不一定是共情能力的缺失
- 职业习惯：某些职业（医生、律师、工程师）习惯用分析性语言，不一定反映人格
- 共情疲劳状态（如照顾者倦怠）可能导致临时性共情空洞

## ▲ D29 优越归因模式 → 暗黑人格（Dark Tetrad-Narcissism）
**置信度：探索性**

### 研究基础

- **自恋的语言标记（Holtzman, 2017）**：自恋者在语言中表现出极端自我服务归因——成功100%内归因+失败100%外归因。使用更多第一人称单数（"我"）、更多绝对化词汇（"总是""绝对"）、更多优越性比较（"比大多数人都……"）
- **Buffardi & Campbell（2008）, *Narcissism and Social Networking***：自恋者在社交媒体中使用更多的自我宣传性语言、更少的集体导向语言
- **事后聪明偏差（Hindsight Bias, Roese & Vohs, 2012）**："我早就知道会这样"类表述与自我强化的认知策略相关——高自恋者更多事后聪明表述
- **普遍性自我服务归因偏差 vs 极端化归因**：正常范围内的自我服务归因（成功归己/失败归外）是普遍的认知偏差——关键区分在于极端程度和跨语境一致性
- **Purdue 2025 中文社交媒体分析**：极端归因模式（成功100%内归+失败100%外归）在中文社交媒体中可被自动检测，与自我报告自恋得分正相关（r≈0.3），但大量"正常自恋"（健康水平的自信表达）会产生假阳性

### 核心映射

| 特征 | 可能的心理指向 |
|------|--------------|
| 成功100%内归因+失败100%外归因，跨语境一致 | 极端自我服务归因→可能与自恋型归因风格相关 |
| 频繁"我早就说过""我早知道"类表述 | 事后聪明偏差→自恋型认知策略 |
| 成功归因带绝对化修饰（"完全是我的功劳"） | 自恋型归因放大 |
| 失败归因带指责（"都是他们的问题""被拖累了"） | 外归因+敌对归因→可能低宜人性+自恋 |
| 归因两极但伴随自嘲/自省信号 | 正常自我服务偏差→非极端化 |
| 谦逊包装的自我归因（"运气好……但确实也付出了很多"） | 社交调控下的自恋表达→可控型 |

### 与D10的关系
D29极端化归因往往伴随D10（绝对化词汇）高频——"完全是我的功劳""绝对是他们的错"。D29+D10跨语境高频组合是更强的信号。

### 局限性
- 正常范围内的自我服务归因偏差（self-serving bias）是普遍认知现象——仅极端化+跨语境一致的归因模式才有心理病理学意义
- 成就导向文本（如面试、绩效自评）天然触发自我归因，不反映人格
- 文化差异：个人主义文化中自我归因更为普遍，集体主义文化中自谦归因更为普遍
- 中文"运气好"有可能是谦逊策略而非真实归因——需要跨语境判断

## ▲ 零样本LLM推估验证层理论（Zero-Shot LLM Estimation）

### 概念

零样本LLM推估验证层（Zero-Shot LLM Estimation）是一种独立于LPP分析之外的**外部效度检验方法**。它让一个独立的LLM仅基于原文文本（零样本——不提供参考标准、不提供LPP分析结果）对心理特质进行1-10评分，然后将结果与LPP分析结论对比。这是对LPP结论的独立外部验证。

### 研究基础

- **Nature Human Behaviour（2026）, *Large Language Models in Psychological Assessment***：在一项大规模跨语言研究中，研究团队让多个LLM（包括Claude、GPT-4等）仅基于自然语言文本（社交媒体帖子、日记条目、书信等）对大五人格进行零样本评分。结果发现：
  - LLM的评分与自陈量表（BFI-2）的相关性达到r=0.45-0.65（因文本长度和特质而异）
  - 开放性（O）的外显行为相关性最高（r≈0.60），宜人性（A）最低（r≈0.45）
  - 单个LLM的评分与人类评分者信度（inter-rater reliability）处于同一水平（ICC≈0.5-0.7）
  - 多个LLM评分的聚合比单个LLM更可靠
  - 当文本长度≥300字时，评分效度显著上升
- **方法论意义**：该研究结果表明LLM能从自然语言中提取有效的人格信号——这不意味着LLM能替代人格测评，而是为基于语言的心理学分析提供了独立验证的可能性

### 与LPP的关系

零样本推估验证层在LPP框架内的定位是**独立的外部效度参考**，而非LPP分析的替代或修正：

| 特性 | LPP分析 | 零样本LLM推估 |
|------|---------|--------------|
| 方法论 | 四步流程（D1-D29→场景定位→归纳→报告） | 单一LLM调用，端到端评分 |
| 粒度 | 每个归纳有原文引用和场景证据链 | 评分+一句理由，缺乏过程透明度 |
| 置信度 | 按信号强度分层（■●▲△） | 统一输出1-10分，无内置置信度 |
| 独立性 | 分析框架→结论 | 盲评（不见LPP结果） |
| 目的 | 提供完整心理画像 | 提供LPP结论的外部验证 |

### 方法论原则

1. **独立性**：推估LLM不应见过LPP分析结果——分析过程中不可将零样本结果作为参考
2. **盲评设计**：推估LLM仅接收原文文本+评分指令，不包含任何背景信息
3. **收敛≠真理**：即使零样本推估与LPP完全收敛，也不意味着结论是"真实"的——只是两个独立方法得出相同结论
4. **分歧≠错误**：两个独立分析方法的分歧是有信息量的——可能反映LPP捕捉到了LLM推估忽略的模式，或LPP过度解读了某些模式

### 在LPP流程中的位置

```
LPP Step 1 → Step 2 → Step 3 → Step 4(草稿)
                                          ↓
                              [独立的] LLM零样本推估 ← 仅原文
                                          ↓
                                    对比LPP分析结果
                                          ↓
                                    收敛→增强置信度
                                    分歧→在报告中呈现
                                          ↓
                                        终稿
```

### 限制

- 零样本LLM推估的效度取决于：LLM型号、提示语设计、文本长度和语境
- LLM在非英语文本上的效度低于英语文本——中文零样本推估的效果尚需更多研究
- LLM可能在评分中受到社会赞许性偏差影响（给出"合理"而非"真实"的评分）
- 当前最佳实践是使用多个LLM型号做推估并取平均，而非依赖单一模型
- 零样本推估不能作为诊断依据——它与LPP一样是概率性推论

## ■ 神经多样性校准层（Step 1.5）理论基础

### 概述

ND层（Step 1.5）的理论前提是：某些语言模式的差异源自神经类型的认知处理差异，而非人格特质——如果不对这些模式做confound校准，大五/大七的归纳结果会系统性偏倚。本层不诊断神经类型，只标记confound方向。

### ND1 感官语言密度 → HSP/ASD confound

**置信度：● 中**

**理论基础**：
- **Aron & Aron（1997）, Sensory Processing Sensitivity（SPS）**：高度敏感者（约占15-20%人口）的特征包括深层信息加工、易过度刺激、高情绪反应性、对细微刺激敏感。SPS在语言中的表现之一是感官描述密度和强度高于基线。
- **感官敏感与情绪性（N）的混淆风险**：高敏感者对感官输入的强烈反应（"太亮了""太吵了"）在语言上可能与高焦虑/高神经质的表达相似——关键在于区分触发源是外部感官超载还是内部情绪困扰。Boterberg & Warreyn（2022）发现ASD和HSP在感官敏感上有显著重叠但机制不同——HSP的感官敏感伴随深度加工，ASD的感官敏感伴随感觉调节障碍。
- **中文语境的特殊性**："敏感"一词在中文中涵盖感官敏感（感官）、情绪敏感（情绪）、社交敏感（人际）三个不同概念——在口语中经常混用。中文文本中"敏感"类自我标签需做进一步的语境归类后才能确定是ND源还是人格源的confound。

**confound机制**：
```
ND1↑ → 感官语言密集 → 看似情绪反应性高 → N↗（误读）
ND1'↑ → 感官寻求 → 看似冲动/低审慎 → A↘（误读）
调节方式：识别触发源是否以外部感官输入为主 → N评分下调
```

**局限性**：
- HSP在中文互联网已被广泛自标签，自我标签不等于真实的高感官敏感
- 隐喻性感官语言（修辞性使用身体意象）不应纳入confound计算
- 感官语言密度可能受话题和语境驱动（如旅行游记中自然更多感官描述）

---

### ND2 系统化加工标记 → ASD confound

**置信度：● 中**

**理论基础**：
- **Baron-Cohen（2002）, Extreme Male Brain / Systemizing Theory**：ASD的核心认知特征之一是极强的系统化（Systemizing）倾向——偏好分析系统规则、对规则的例外不耐受、用"如果-那么"规则理解世界。系统化在语言中的体现是：高频率的规则陈述语言（"按理说""应该""这说不通"）、对模糊和非结构化的低耐受。
- **系统化与开放性的关系**：系统化加工者倾向于偏好结构化、可预测、规则明确的信息环境——这与低开放性（O）的语言表达有显著重叠。Lodi-Smith et al.（2019）发现ASD自评者在大五上的典型特征是低开放性、低外向性（部分源于社交差异而非社交动机）、高尽责性（源于规则遵从）。ND2标记的核心目标是区分"ASD源的规则驱动式思维"与"人格源的低开放性"。
- **ASD与宜人性（A）的误读风险**：ASD个体的"直率""不懂社交暗示""规则先于人情"可能在语言上被误读为低宜人性。实际机制可能不同——这是社交认知差异（无法解读社交线索）而非敌意动机。这在中文语境中尤其容易被误读（中文高语境文化对社交线索敏感性有较高期望）。

**confound机制**：
```
ND2↑ → 系统化语言密集 → 看似偏好确定/规则 → O↘ + T↘
调节方式：区分"规则遵从（人格）"vs"系统化思维（认知风格）"→ 前者调O/T，后者不调
```

**与D35/D39的边界**：
- D35（评价框架）：测量褒贬标签选择——正常人格层面的评价倾向
- D39（隐喻源域）：测量隐喻来源偏好——认知风格层面
- ND2：仅在系统化语言表现出**非灵活性**（对规则的执著超出合理范围、对所有话题都用系统化解释）时才激活

---

### ND3 注意力调节标记 → ADHD confound

**置信度：● 中**

**理论基础**：
- **ADHD的语言特征（Engelhardt et al., 2017; 2020）**：ADHD个体在自发言语中表现出更多话段转换、更多填充词（"嗯""那个"作为组织思想的时间购买）、更多不完整句。在书面文本中表现为话题跳跃更快、叙事结构非线性、执行功能相关的词汇更多（"拖延""忘了""又……了"）。
- **ADHD与尽责性（C）的混淆风险**：ADHD的执行功能差异（时间管理困难、组织困难、拖延）在语言中可能被误读为低尽责性。Romo-Nava et al.（2022）发现ADHD个体在大五上的低尽责性与执行功能缺陷高度相关（r≈0.6），但两者在机制上不同——前者是"动机/自律问题"，后者是"执行功能差异"。语言中的关键区分标记是：ADHD源的低尽责表达伴随"想做好但做不到"的挫败感，人格源的低尽责表达伴随"无所谓"的漠然。
- **ADHD与情绪性（N）的混淆风险**：ADHD的情绪失调（情绪反应强度大、恢复慢）可能被误读为高神经质。关键区分：ADHD情绪失调是**情境触发的反应强度**（触发物消失后情绪快速回落），高N的情绪反应是**基线水平高**（即使在无触发时也有焦虑/负面倾向）。
- **超聚焦（Hyperfocus）**：ADHD的注意力调节是双极的——分散注意力和超聚焦都是同一机制的两端。超聚焦描述（"停不下来""忘了时间""太投入了"）是ND3的重要标记之一。

**confound机制**：
```
ND3↑ → 注意力切换快 → 看似随意/无条理 → A↘（行事风格低）
ND3'↑ → 超聚焦语言 → 看似专注但不一致 → A评分一致性降级
调节方式：区分"执行功能差异（ND）"vs"低自律（人格）"
```

**中文语境特殊校准**：
- "拖延"一词在中文互联网被普遍自标签——单次提及不足为信
- 需≥2个不同语境中的执行功能描述（如时间管理+组织困难+任务启动困难）

---

### ND4 社交脚本与masking标记 → ASD confound（社交维度）

**置信度：● 中**

**理论基础**：
- **ASD社交masking（Hull et al., 2017; Pearson, 2023）**：ASD个体在社交场合中有意识地伪装自身社交行为以符合神经典型期望——包括模仿表情、排练对话、隐藏自我刺激行为。Masking在语言中的标记包括：社交表演意识（"装""演""假装正常""学别人"）、社交疲劳（"社交之后就累了""需要独处充电"）、对社交规则的外显分析和困惑（"我不懂为什么要寒暄""社交规则太复杂了"）。
- **Masking与外向性（E）的误读风险**：高masking意识的ASD个体在外向上可能有两种模式——一种是"看起来高外向"（因为努力学习社交脚本并成功执行），另一种是"看起来极端内向"（因为社交疲劳导致回避）。两者都不是真实的外向性水平。不区分的后果是masking行为被误读为真实的外向或内向。
- **Masking与依恋回避的误读风险**：ASD源的社交差异（回避眼神、回避亲密因不理解社交规则）可能与依恋回避（因恐惧亲密而回避）在语言上相似。关键区分：ASD源的社交回避在**所有**社交关系中一致出现（包括安全关系），依恋回避在**亲密关系**中更明显。

**confound机制**：
```
ND4↑ → masking语言 → 看似低社交动机 → E↘（误读为内向）
ND4'↑ → 社交规则外显分析 → 看似社交认知≠社交动机 → E不降
调节方式：区分社交疲劳（ND）vs低社交动机（人格）
```

**与D35的边界**：
- D35（评价框架）测量褒贬标签选择——是社会认知层面的观察
- ND4仅在明确提到"装/演/假装/学别人"或社交疲劳的有意识体验时才激活
- 社交回避如果不伴随对masking的意识，更可能是人格/依恋源而非ND源

---

### ND5 认知补偿与替代通路 → 阅读障碍/ND优势 confound

**置信度：▲ 弱**

**理论基础**：
- **阅读障碍的认知补偿（Shaywitz & Shaywitz, 2005; 2020）**：阅读障碍者的语言系统在音韵加工方面存在核心缺陷，但通过发展替代认知通路（视觉空间思维、系统化推理、叙事记忆）来补偿。这些补偿优势在语言中留下痕迹——倾向于用图像/空间关系思考、偏好系统化/模式化解释而非线性叙事、对故事有出色的整体记忆但细节回忆困难。
- **阅读障碍与开放性（O）的关系**：许多阅读障碍者在语言上表现出高开放性——不是源于传统的经验开放性，而是源于发展出非语言认知通路后的另类思维方式。"图像思维"与"高开放性"在语言上的表现有显著重叠（隐喻使用、非常规联想、发散思维）。
- **对I（才干）的误读风险**：阅读障碍者可能在书面表达中词汇丰富度/句法复杂度的"书面能力"低于其实际认知能力——这会被误读为"低I（才干）"。McCardle & Miller（2017）发现阅读障碍者在口头叙事中的复杂度和深度显著高于其书面写作中的表现——仅分析书面文本会系统性低估认知能力。
- **LPP的校准建议**：在中文语境中，阅读障碍认知度极低——不依赖自我标签，仅从替代通路痕迹（如视觉/空间思维语言的丰富度、系统化模式识别能力的表现）反向读取。

**confound机制**：
```
ND5↑ → 视觉/空间思维语言 → 看似非文字能力强但书面表达弱 → I↘（误读）
ND5↑ → 系统化模式识别 → 看似高认知复杂度 → O↗（方向一致，不调）
调节方式：区分"认知补偿通路"vs"真实认知能力"→ I不降或上调
```

---

### ND6 时间处理标记 → ADHD/ASD confound

**置信度：▲ 弱**

**理论基础**：
- **时间盲（Time Blindness, ADHD）**：ADHD个体在时间感知上的差异——"主观时间"脱离时钟时间——表现为时间估计不准、无法感知"时间的流逝"、"感觉才五分钟结果已经过了两小时"。Time Blindness的语言标记包括：对时间流逝的惊讶（"居然已经这个点了"）、时间锚点的模糊、对时间管理的持续挫败。
- **时序组织困难（ASD）**：ASD个体可能在顺序组织和事件时间排序上存在困难——在自传叙事中表现为时间线索的跳跃或混乱（"理不清先后"）。这不同于ADHD时间盲——ASD的时间困难更多是**序列组织**（事件在时间轴上排序），ADHD的时间困难更多是**时间感知**（时间流逝速度的主观体验）。
- **时间处理差异与尽责性（C）的误读风险**：时间盲的语言表达（"又迟到了""不知道过了多久""来不及了"）在浅层阅读中可能被误判为低尽责性（时间管理不好=不负责）。但关键区分在于：ADHD源的时间困难伴随**对时间管理的高度焦虑和失败体验**（"我又迟到了，我真的很努力了但还是……"），而低尽责性的时间随意则伴随**漠然**（"迟到就迟到呗"）。
- **中文语境校准**：中国高压社会（早八、996、deadline驱动文化）下，几乎所有人都有时间焦虑——区分情境性时间焦虑（由外部deadline驱动）vs结构性时间盲（无论是否有外部压力都难以感知时间）。

**confound机制**：
```
ND6↑ → 时间盲语言 → 看似不守时/不可靠 → A↘（行事风格低）
调节方式：区分"时间感知差异（ND）"vs"不负责（人格）"→ A评分上调或不调
```

### ND↔ND方向冲突规则

**定义**：当两个独立的ND标记对同一人格维度产生相反的confound方向时，需要冲突解决规则：

| 冲突对 | confound方向冲突 | 解决规则 | 优先级逻辑 |
|--------|-----------------|---------|-----------|
| ND1↑（感官密集→N↗）vs ND1'↑（感官寻求→A↘） | 不同维度，无直接冲突 | 各自独立调节 | 非同一维度时各自执行 |
| ND2↑（系统化→O↘）vs ND5↑（补偿通路→O↗） | O维度，双向对立 | **ND5不调节，ND2部分调节** | 认知补偿（ND5）是次级策略，不优于系统化风格（ND2）的证据权重 |
| ND3↑（注意切换→A↘）vs ND3'↑（超聚焦→A局部↗） | A维度，方向相反 | **取ND3↑为准，ND3'↑仅降一致性** | 超聚焦（ND3'）是情境性的（仅兴趣领域），整体注意力调节（ND3↑）更稳定 |
| ND1↑（HSP感官密集→N↗）vs ND4↑（masking→E↘） | 不同维度，无直接冲突 | 各自独立调节 | 非同一维度 |
| ND3↑（ADHD注意切换→A↘）vs ND6↑（时间盲→A↘） | A维度，方向一致 | **叠加调节**（两个独立ND标记指向同一方向） | 两个独立机制→confound证据增强→调节幅度加 |

**通用优先级准则**：
1. **信号强度优先**：■强（7-9分）的ND标记覆盖●中（4-6分）的ND标记
2. **直接证据优先**：有跨语境行为描述的标记 > 仅有自我标签的标记
3. **稳定模式优先**：跨语境一致的标记 > 单一语境激活的标记
4. **叠加标记增强**：≥2个独立ND标记指向同一confound方向 → 调节幅度增加50%

### ND输出层的报告呈现规则

ND分析结果对第一部分叙事的影响：
- ND校准后的人格差异**可以**出现在第一部分的描述中，但需遵循以下规则：
  - **不出现ND技术术语**：不写"你可能存在ADHD时间盲"——写"你的时间感知方式与他人不同——对你来说'很快'和'很久'的体验可能和时钟不一致"
  - **以优势+代价框架呈现**：将ND特征描述为认知差异而非缺陷——"你不是不守时，而是你对时间的体验与时钟不同步"
  - **仅在ND信号≥●中（4-6分）时在叙事中使用ND校准视角**：弱信号（▲）时不在第一部分暗示任何ND源解释
  - **先人格后ND**：第一部分的基线描述基于人格归纳结果，ND调节仅用于对矛盾处提供替代解释——"这让你在[人格特征]的基础上，还有一个特点：……"
- **审计检查项**：第一部分的ND相关描述是否在第二部分ND层中有对应的模式证据？→ 无证据则删

### ND层的跨时间稳定性

纵向对比（同一人不同时间点的文本）中ND confound的处理规则：
- **ND信号的时间一致性**：同一人在不同时间点文本中的ND信号强度可能变化——时间处理标记（ND6）可能在高压力时期增强，感官语言密度（ND1）可能在不同感官环境中变化
- **跨时间confound变化处理**：
  - ND信号跨时间一致 → 使用统一校准幅度
  - ND信号强度跨时间波动 → 以中位信号强度为准，标注"该ND标记在不同时间点强度波动，校准以中位值为准"
  - ND信号在某些时间点消失 → 检查是情境触发（如特定压力源）还是真实变化，仅对激活的时间点做校准
- **报告中的呈现**：在纵向对比报告中，各时间点的ND校准独立执行后，对比"校准前vs校准后"的人格变化——如果ND校准后的变化趋势不同于校准前，标注"ND confound在跨时间对比中起主要作用"

### ND层的重要限制

1. **不能用作诊断**：ND层识别的是"语言模式可能反映神经类型差异"——不提供诊断。ASD/ADHD/HSP/阅读障碍的诊断需要临床评估
2. **文化校准敏感**：ND标记在不同文化中的baseline不同——中文语境下情感夸张表达可能是修辞习惯而非感官超载
3. **自我标签不可靠**：中文互联网的自诊文化下，ASD/ADHD/HSP自我标签的实际prevalence远高于临床——需以行为描述（语言模式）而非标签为准
4. **共病复杂**：ND各亚型之间存在高共病（ADHD+ASD~50-80%），单一标记可能反映共病而非单一亚型
5. **隐喻性感官语言减半**：中文写作中大量的身体意象（"心如刀割""燃烧"）是修辞传统——不应全额计入ND1

### ND × 情绪轨迹交叉验证规则

情绪轨迹分析（见SKILL.md"非D分析层"）的结果可以反过来为ND confound提供调节信号——情绪轨迹的类型和特征有助于区分某个ND标记是**稳定神经类型特质**还是**状态性反应**。

#### 交叉验证路径

| ND标记 | 情绪轨迹中的信号 | 对ND confound的调节作用 |
|-------|----------------|----------------------|
| **ND1↑（感官密集→N↗）** | 轨迹呈**平坦型**（情绪无波动）但感官语言持续密集 → 感官加工风格独立于情绪状态 → **ND1 confound保持权重** |
| **ND1↑（感官密集→N↗）** | 轨迹呈**下降型**或**反刍循环型**且感官语言集中在负面段 → 感官描述可能由情绪困扰驱动而非认知风格 → **ND1 confound降低一级**（从●中→▲弱/▲弱→不激活） |
| **ND2↑（系统化→O↘）** | 轨迹呈**平坦型**（无论情绪如何都保持规则提取语言）→ 系统化是稳定认知风格 → **ND2 confound保持** |
| **ND2↑** | 轨迹呈**波动型**且系统化语言在焦虑段落增强（压力→规则→安抚）→ 系统化是情绪调节策略而非认知风格 → **ND2 confound改为"情境性"标注** |
| **ND3↑（注意切换→A↘）** | 轨迹呈**波动型**+**高频率话题跳跃**（段落间情绪无明显关联但话题频繁切换）→ ADHD注意力切换独立于情绪 → **ND3 confound保持** |
| **ND3↑** | 轨迹呈**下降型**且话题跳跃随情绪恶化而加剧 → 注意力切换由情绪驱动 → **ND3 confound降低一级，改为"情绪性注意分散"标注** |
| **ND4↑（masking→E↘）** | 轨迹呈**平坦型**+**全篇高礼貌/高社交脚本密度** → masking为稳定社交风格 → **ND4 confound保持** |
| **ND4↑** | 轨迹呈**上升型**（从抑制到放松）且masking标记在文本后半减少 → 语境适应而非稳定masking → **ND4 confound降低一级** |
| **ND6↑（时间盲→A↘）** | 轨迹呈**无时间结构型**（全篇无时间参照/时序混乱）+**反刍循环型** → 时间处理困难独立于情绪还是由情绪反刍引发 → **需结合ND6具体引用判断**：时间盲描述在情绪段落内外一致→保持；仅出现在负面段落→降低 |

#### 交叉验证后的confound决策树

对每个ND标记的激活，在完成情绪轨迹交叉验证后执行以下决策逻辑：

```
输入：ND标记激活（如ND1↑ → N↗，中信号）
      + 情绪轨迹类型（如下降型）
      + 该标记与情绪轨迹的时间耦合度（高/中/低）

Step 1：检查耦合度
  ├─ 低耦合（ND标记均匀分布，独立于情绪波动）
  │   → ND confound保持原权重
  ├─ 中耦合（ND标记在正面/负面段落有差异但非完全一致）
  │   → ND confound降半级（●中→●中/▲弱边界）
  └─ 高耦合（ND标记集中在情绪负面段落）
      → 执行 Step 2

Step 2：区分方向（高耦合时）
  ├─ ND标记在负面段增强 → 可能是情绪引起的次级反应
  │   → ND confound降低一级（■强→●中，●中→▲弱，▲弱→不激活）
  └─ ND标记在正面段也出现（但强度低于负面段）
      → 部分独立于情绪，但情绪可加重表现
      → ND confound降低半级

Step 3：报告标注
  经情绪轨迹交叉验证调节后的confound在报告中标注：
  - "经情绪轨迹交叉验证，该ND标记与情绪轨迹[高/中/低]耦合，
    confound置信度从[原]调节至[现]"
```

#### 在拼凑样本中的处理

拼凑样本中情绪轨迹分析跳过整体曲线，改做碎片独立效价标记。此规则同样适用于ND × 情绪轨迹交叉验证：

- 对每个碎片独立执行ND × 情绪轨迹交叉验证
- 各碎片的ND confound调节结果独立记录
- 跨碎片取ND confound强度的中位值作为最终校准幅度
- 碎片间ND confound强度差异 > 1个等级（如碎片A为■强、碎片B为不激活）→ 标注"该ND标记在不同碎片中confound强度不一致，校准以中位值为准"
